22. Sebanyak 95 siswa diberikan angket untuk menulis aplikasi handphone yang dimiliki. Setelah dikumpulkan, diketahui 75 siswa memiliki aplikasi game, 60 siswa memiliki aplikasi rumah belajar, 50 siswa memilih aplikasi Game dan rumah belajar, serta sisanya tidak memiliki kedua aplikasi. Tentukan banyak siswa yang tidak memiliki aplikasi Game atau rumah belajar ?bantu dong ​


Penjelasan dengan langkah-langkah:

Semua siswa = S

Aplikasi game = a

Aplikasi rumah belajar = b

n (S) = 95

n (a) + n (b) - n (anb) + n (aub)' = 95

75 + 60 - 50 + n (aub)' = 95

85 + n (aub)' = 95

n (aub)' = 95 - 85

n (aub)' = 10

Banyak siswa yang tidak memiliki aplikasi game atau rumah belajar = 10 siswa
