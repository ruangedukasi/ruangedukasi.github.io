Suara Puisi: Suara puisi adalah unsur yang berkaitan dengan nada dan irama yang dihasilkan oleh puisi. Suara puisi sering kali memberikan efek yang kuat bagi pembaca.

Makna Puisi: Makna puisi adalah pesan atau makna yang ingin disampaikan oleh penyair melalui puisinya. Makna puisi seringkali bersifat abstrak dan tersembunyi, sehingga memerlukan pemahaman yang mendalam bagi pembaca.

Passive voice adalah kalimat yang subjeknya dikenai suatu pekerjaan atau tindakan dari kata kerjanya, artinya bukan subjek tersebut yang melakukan suatu aksi.

Rumus Passive VoiceSimple present tense

Active Voice : S + V1 + O

Passive Voice: S + to be (am, are, is) + V3 + by + O

Present continuous tense

Active Voice : S + to be (am, are, is) + V1-ing + O

Passive Voice: S + to be (am, are, is) + being + V3 + by + O

Present perfect tense

Active Voice : S + to be (am, are, is) + being + V3 + by + O

Passive Voice: S + have/has + been + V3 + by + O

Present perfect continuous tense

Active Voice : S + have/has + been + V1-ing+ O

Passive Voice: S + have/has + been + being + V3 + by + O

Simple past tense

Active Voice : S + V2 + O

Passive Voice: S + to be (was/were) + V3 + by + O

Past continuous tense

Active Voice : S + to be (was, were) + V1-ing + O

Passive Voice: S + to be (was/were) + being + V3 + by + O

Past perfect tense

Active Voice : S + had + V3 + O

Passive Voice: S + had + been + V3 + by + O

Past perfect continuous tense

Active Voice : S + had + been + V1-ing + O

Passive Voice: S + had + been + being + V3 + by + O

Simple future tense

Active Voice : S + will not + V1 + O

Passive Voice: S + will be + V3 + by + O

Future continuous tense

Active Voice : S + will + be + V1-ing + O

Passive Voice: S + will + be + being + V3 + by + O

Future perfect tense

Active Voice : S + will + have + V3 + O

Passive Voice: S + will + have + been + V3 + by + O

Future perfect continuous tense

Active Voice : S + will + have + been + V1-ing + O

Passive Voice: S + will + have + been + V1-ing + O

Passive Voice: S + would + be + V3 + by + O

Active Voice : S + would + be + V1-ing + O

Passive Voice: S + would + be + being + V3 + by + O

Past future perfect tense

Active Voice : S + would + have + V1 + O

Passive Voice: S + would + have + been + V3 + by + O

Past future perfect continuous tense

Active Voice : S + would + have + been + V1-ing + O

Passive Voice: S + would + have + been + being+ V3 + by + O

Modals (present)

Active Voice : S + may/can/must + V1 + O

Passive Voice: S + may/can/must + be + V3 + by + O

Modals (past)

Active Voice : S + might/could/had to + V1 + O

Passive Voice: S + might/could/had to + be + V3 + by + O

Infinitive

Active Voice : S + has to/have to + V1 + O

Passive Voice: S + has to/have to + be + V3 + by + O

Going to

Active Voice : S + to be (am, are, is) + going to + V1 + O

Passive Voice: S + to be (am, are, is) + going to + be + V3 + by + O
