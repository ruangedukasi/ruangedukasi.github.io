4. Tentukan akar persamaan kuadrat berikut dengan 3 cara yang telah kalian pelajari

a. x² - 1 = 0b. 4x² + 4x + 1 = 0c. - 3x² - 5x + 2 = 0d. 2x² - x - 3 = 0e. x² - x + ¼ = 0

Jawab :

a. x² - 1 = 0     (Memfaktorkan)

x² - 1 = 0

(x + 1)(x - 1) = 0

x + 1 = 0 atau x - 1 = 0

x = -1 atau x = 1

HP = {-1, 1}

4x² + 4x + 1 = 0

(x + ¹/₂)² = -¹/₄ + (¹/₂)²

(x + ¹/₂)² = -¹/₄ + ¹/₄

(x + ¹/₂)² = 0

x + ¹/₂ = 0

x = -½

HP = -½

c. - 3x² - 5x + 2 = 0   (Memfaktorkan)

- 3x² - 5x + 2 = 0  (dikali negatif)

3x² + 5x - 2 = 0

(3x - 1) (x + 2) = 0

3x - 1 = 0

    x₁ = 1/3

atau

x + 2 = 0

HP = {-2, 1/3}

Pembahasan

Seni lukis adalah salah satu bentuk turunan dalam seni rupa yang akan menghasilkan karya seni rupa dua dimensi. Dalam suatu lukisan, ada beberapa aliran yang bisa digunakan oleh seorang pelukis dalam melukis karya seni yang akan dibuatnya.

Berikut adalah beberapa aliran dalam seni lukis:

Romantisme: aliran dalam seni lukis yang menggunakan warna yang kontras dan dinamis untuk mendapatkan kesan yang dramatis pada lukisannya.Realisme: aliran dalam seni lukis yang tema lukisannya berasal dari realita kehidupan yang senyata-nyatanya.Impresionisme: aliran dalam seni lukis yang menggambarkan kesan warna yang ada pada suatu objek sehingga menghasilkan karya seni lukis yang cenderung tidak detail dan kabur.Kubisme: aliran dalam seni lukis yang menggunakan bentuk geometris untuk membentuk hasil lukisannya.

P’(2, 2)Q’(1, 4)R’(3, 4)S’(4, –2)

Rotasi merupakan transformasi geometri untuk mencari bayangan suatu objek jika diputar sejauh α terhadap titik tertentu. Jika diputar searah jarum jam maka sudutnya (–α), sedangkan jika diputar berlawanan arah jarum jam sudutnya (+α).

Bayangan dari titik (x, y) jika diputar 90˚ dengan pusat (0, 0)

berlawanan arah jarum jam adalah (–y, x).searah jarum jam adalah (y, –x).

berlawanan arah jarum jam adalah (b – y + a, x – a + b).searah jarum jam adalah (y – b + a, a – x + b).

Penjelasan dengan langkah-langkah

2. Diketahui

Segiempat PQRS diputar 90˚ berlawanan arah jarum jam terhadap titik asal dengan

P(2, –2)Q(4, –1)R(4, –3)S(2, –4)

Ditanyakan

Gambarlah segiempat PQRS tersebut beserta bayangannya!

Jawab

Langkah 1

Bayangan titik (x, y) jika diputar 90˚ berlawanan arah dengan pusat titik asal adalah

(–y, x)

Berarti bayangan dari titik:

P(2, –2) adalah P’(–(–2), 2) = P’(2, 2)Q(4, –1) adalah Q’(–(–1), 4) = Q’(1, 4)R(4, –3) adalah R’(–(–3), 4) = R’(3, 4)S(2, –4) adalah S’(–(–4), 2) = S’(4, 2)
