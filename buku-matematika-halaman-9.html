4. Tentukan akar persamaan kuadrat berikut dengan 3 cara yang telah kalian pelajari

a. x² - 1 = 0b. 4x² + 4x + 1 = 0c. - 3x² - 5x + 2 = 0d. 2x² - x - 3 = 0e. x² - x + ¼ = 0

Jawab :

a. x² - 1 = 0     (Memfaktorkan)

x² - 1 = 0

(x + 1)(x - 1) = 0

x + 1 = 0 atau x - 1 = 0

x = -1 atau x = 1

HP = {-1, 1}

b. 4x² + 4x + 1 = 0   (Melengkapi kuadrat sempurna)

4x² + 4x + 1 = 0

x² + ⁴/₄ x + (⁴/₈)² = -¹/₄ + (⁴/₈)²

(x + ¹/₂)² = -¹/₄ + (¹/₂)²

(x + ¹/₂)² = -¹/₄ + ¹/₄

(x + ¹/₂)² = 0

x + ¹/₂ = 0

x = -½

HP = -½

c. - 3x² - 5x + 2 = 0   (Memfaktorkan)

- 3x² - 5x + 2 = 0  (dikali negatif)

3x² + 5x - 2 = 0

(3x - 1) (x + 2) = 0

3x - 1 = 0

    x₁ = 1/3

atau

x + 2 = 0

    x₂ = -2

HP = {-2, 1/3}

d. 2x² - x - 3 = 0  (memfaktorkan)

2x² - x - 3 = 0

(2x - 3)(x +1) = 0

x = 3/2 ataux = -1

aⁿ × aᵐ = aⁿ⁺ᵐaⁿ : aᵐ = aⁿ⁻ᵐ(aⁿ)ᵐ = aⁿᵐ(ab)ⁿ = aⁿ × bⁿ

Pak Asep memiliki sebuah kolam renang berbentuk silinder di belakang  rumahnya. Diameter kolam tersebut adalah  14√3  meter dengan kedalaman  150√2  cm. Apabila Pak Asep ingin mengisi kolam tersebut sampai penuh,  berapa liter air yang dibutuhkan oleh Pak Asep? Tuliskan jawabanmu dalam  bentuk perpangkatan paling sederhana.

Sifat Perpangkatan

xᵃ × xᵇ = xᵃ⁺ᵇxᵃ : xᵇ = xᵃ⁻ᵇ(xᵃ)ᵇ = xᵃᵇ(xy)ⁿ = xⁿ . yⁿ

Untuk lebih lengkap sifat perpangkatan bisa dilihat pada lampiran

Rumus volume tabung

[tex]\boxed {V = \pi r^{2} t}[/tex]

Pembahasan

Diketahui :

Kolam berbentuk silinder (tabung)

diameter = 14√3 m

kedalaman (t) = 150√2 cm

Ditanya :

Volume kolam sampai penuh dalam satuan liter?

Jawab :

Karena yang diminta volume air kolam dalam liter, maka liter = dm³. Sehingga satuan meter dan cm kita ubah ke dalam dm.

Diameter = 14√3 m = 140√3 dm

Jari-jari (r) = 140√3 dm : 2 = 70√3 dm

kedalaman (t) = 150√2 cm = 15√2 dm

Menentukan volume air kolam

V = π × r² × t

  = ²²/₇ × (70√3)² × 15√2 dm³

  = ²²/₇ × 4900 × 3 × 15√2 dm³

  = 22 × 700 × 3 × 15√2 dm³

  = 693.000√2 dm³

  = 693.000√2 liter

Jadi air yang dibutuhkan oleh pak Asep adalah 693.000√2 litar.

Garis y = -x              A (x, y) → A' (-y , -x)

Titik (a, b)                 A (x, y) → A' (2a - x , 2b - y)
