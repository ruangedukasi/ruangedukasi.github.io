12. Contoh soal rotasi dalam geografi


Jawaban:

Tentukan bayangan garis y = 5x + 4 oleh rotasi R(O, -90)!

a. x - 5y – 4 = 0

b. x + 5y + 4 = 0

c. 5x + 5y – 4 = 0

d. 5x - 5y – 4 = 0

e. x + 5y – 4 = 0

*Pembahasan* :

(x, y) o (y, -x)

x’ = y , y’ = -x

x’ = 5(-y’) + 4

x’ + 5y’ – 4 = 0 Jadi bayangan x + 5y – 4 = 0

38. contoh soal prinsip geografi


Dalam kurun waktu lima tahun terakhir, wilayah pantai Barat Sumatera sudah beberapa kali diguncang gempa tektonik dan dilihat dari struktur geologinya wilayah tersebut berada di zona tumbukan lempeng.

Prinsip geografi yang digunakan untuk mengkaji fenomena tersebut adalah ....

A. prinsip persebaran

B. prinsip distribusi

C. prinsip interelasi

D. prinsip deskripsi

E. prinsip korologi

Pembahasan

Pada soal hanya mengungkapkan Pantai Barat Sumatra rawan gempa akibat posisinya di zona tumbukan lempeng. Jadi, prinsip yang digunakan adalah prinsip interrelasi.

Jawaban: C


39. soal kemololan dan fraksi Mol pt3​


Penjelasan:

3. fraksi mol => X menyatakan perbandingan jumlah zat terlarut dengan jumlah zat total pelarut + terlarut

x1 = n1 / ( n1 + n2 )

x2 = n2 / ( n1 + n2 )

x1 + x2 = 1

x1 = fraksi mol terlarut

n1 = mol zat terlarut

x2 = fraksi mol pelarut

n2 = mol zat pelarut

4. a. Mr NaOH = Ar Na + Ar O + Ar H

=> 23 + 16 + 1

=> 40

mol NaOH = 20 / 40 => 0,5 mol

Mr H2O = ( 2x Ar H ) + Ar O

=> ( 2 x 1 ) + 16

=> 18

massa H2O = 36 ml x 1 gram / ml = 36 gram

mol H2O = 36 / 18 => 2 mol

b. fraksi mol NaOH: X2 = n2 / ( n1 + n2 )

X2 = 0,5 / ( 2 + 0,5 )

X2 = 0,5 / 2,5

X2 = 0,2
