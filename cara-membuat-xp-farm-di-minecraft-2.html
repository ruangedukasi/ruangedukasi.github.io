15. cara menginstal windows xp lengkap


Berikut ini merupakan tahapan dalam menginstal windows xp :

Pertama-tama, nyalakan komputer atau laptop kemudian tekan DEL pada komputer atau F2 pada laptop. Hal ini dilakukan untuk masuk ke setup menu serta memastikan "booting windows" dimulai dari CD.Masuk ke setup menu pengaturan "booting" dengan tampilan berbeda, tergantung pada merk masing-masing. Kemudian cari "boot" untuk "setting booting" pertama ke CD "installer windows XP". Arahkan CD ke posisi yang pertama, untuk memposisikan CD ke yang pertama biasanya terdapat keterangannya di samping kiri maupun bawahnya. Kemudian Exit dan Save, kemudian klik yes, maka komputer laptop akan segera booting. Namun perlu diingat, jangan sampai lupa memasukkan CD Windows XP ke komputer atau laptop sebelum restart lagi.Tekanlah tombol apasaja pada saat CMD muncul agar dapat masuk ke proses instalasi windows XP. Waktu yang diberikan untuk menekan tombol hanyalah 5 detik saja sehingga apabila tidak ditekan maka proses instalasi tidak akan terjadi.Masuk ke proses instalasi, pada saat ini tidak melakukan apapun dan kita hanya dapat mengamatinya saja.Tekan Enter untuk memulai proses instalasi.Tekan F8 untuk menyetujui proses instalasi.Kemudian masuk ke pengaturan partisi atau disebut juga pengaturan hardisk atau hard drive.Kemudian untuk meyakinkan penghapusan tekan L.Arahkan kursor ke partisi yang akan diinstal windows XP.Ketikan angka yang diinginkan atau tekan tombol C maupun tekan enter diperbolehkan jika ingin melewatinya.Tekan enter dan proses instalasi dimulai secara otomatis.Setelah melakukan langkah diatas, komputer atau laptop akan melakukan restart secara otomatis. Jangan menekan tombol apapun pada saat CMD muncul.Pada proses penyempurnaan instalasi windows XP, kita tidak perlu melakukan apapun.Setelah proses instalasi selesai, Ikuti langkah-langkah "Personalize name" sesuai data diri dan Setting windows XP sesuai dengan kode atau "product key" yang telah dipersiapkan. Windows XP adalah sistem operasi berbasis grafis yang dibuat oleh Microsoft untuk digunakan pada komputer pribadi, yang mencakup komputer rumah dan desktop bisnis, laptop, dan pusat media (Media Center). Nama "XP" adalah kependekan dari "Experience".

Microsoft sendiri terus menjual Windows XP melalui Custom-built PC (OEM kecil yang menjual komputer rakitan) sampai dengan 31 Januari 2009. Windows XP mungkin akan tetap tersedia bagi para pengguna korporasi dengan volume licensing, sebagai sarana downgrade untuk komputer-komputer yang belum siap menjalankan sistem operasi baru, Windows Vista Business Edition atau Ultimate Edition atau Windows 7 Professional.

.

a=Basis

n=Pangkat

[tex] {a}^{x} \div {b}^{x} = {( \frac{a}{b} )}^{x} [/tex]

[tex] {a}^{ - x} = \frac{1}{ {a}^{x} } [/tex]

.

[tex] {x}^{3} - {y}^{3} = (x - y) \times ( {x}^{2} + xy + {y}^{2} )[/tex]

.
