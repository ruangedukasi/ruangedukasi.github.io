13. matematika hal 93 kelas 8 kurikulum 2013


Lengkapi tabel berikut halaman 93 Buku Matematika kelas 8 smp semester 2

Pada bab ini kita masih membahas soal Lingkaran, yaitu tentang sudut pusat, panjang busur dan Luas Juring

Sudut pusat adalah sudut yang dibentuk oleh dua jari-jari yang berpotongan pada pusat lingkaran

Busur adalah garis lengkung yang menghubungkan dua titik sembarang pada lengkungan lingkaran

Juring adalah daerah di dalam lingkaran yang dibatasi oleh dua jari-jari lingkaran dan busur lingkaran dihadapan sudut pusat yang dibentuk oleh kedua jari-jarinya

Rumus :

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

Busur = [tex]\frac{\theta}{360}[/tex]K

Dimana :

[tex]\theta[/tex] = sudut juring dalam derajat

K = keliling lingkaran (K = 2πr )

L = Luas lingkaran (L = πr² )

Pembahasan :

1. a. Diketahui :

[tex]\theta[/tex] = 90°

r = 7cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : Panjang Busur ?

Dijawab :

Keliling = 2πr

= 2 x [tex]\frac{22}{7}[/tex] x 7

= 44 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

= [tex]\frac{90}{360}[/tex] x 44

= 11 cm


b. Diketahui :

[tex]\theta[/tex] = 60°

r = 21cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : Panjang Busur ?

Dijawab :

Keliling = 2πr

= 2 x [tex]\frac{22}{7}[/tex] x 21

= 132 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

= [tex]\frac{60}{360}[/tex] x 132

= 22 cm


c. Diketahui :

[tex]\theta[/tex] = 120°

Panjang Busur = 88 cm

π = [tex]\frac{22}{7}[/tex]

Ditanya : r ?

Dijawab :

Busur = [tex]\frac{\theta}{360}[/tex]K

88 = [tex]\frac{120}{360}[/tex] x K

88 = [tex]\frac{1}{3}[/tex] x K

K = 264 cm

Keliling = 2πr

264 = 2 x [tex]\frac{22}{7}[/tex] x r

264 = [tex]\frac{44}{7}[/tex] x r

r = 42 cm


d. Diketahui :

r = 100 cm

Panjang Busur = 31,4 cm

π = 3,14

Ditanya : [tex]\theta[/tex] = ?

Dijawab :

Keliling = 2πr

= 2 x 3,14 x 100

= 628 cm

Busur = [tex]\frac{\theta}{360}[/tex]K

314 = [tex]\frac{\theta}{360}[/tex] x 628

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{314}{628}[/tex]

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{1}{2}[/tex]

[tex]\theta[/tex] = [tex]\frac{360}{2}[/tex]

[tex]\theta[/tex] = 180°


e. Diketahui :

[tex]\theta[/tex] = 72°

Panjang Busur = 1.256 cm

π = 3,14

Ditanya : r ?

Dijawab :

Busur = [tex]\frac{\theta}{360}[/tex]K

1.256 = [tex]\frac{72}{360}[/tex] x K

1.256 = [tex]\frac{1}{5}[/tex] x K

K = 6.280 cm

Keliling = 2πr

6.280 = 2 x 3,14 x r

6.280 = 6,28r

r = 1000 cm


2. a. Diketahui :

[tex]\theta[/tex] = 100°

r = 6cm

π = 3,14

Ditanya : Luas Juring ?

Dijawab :

L = πr²

L = 3,14 x 6 x 6

L = 113,04 cm²

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

= [tex]\frac{100}{360}[/tex] x 113,04

= 31,4 cm²


b. Diketahui :

[tex]\theta[/tex] = 25°

Luas juring = 31,4 cm²

π = 3,14

Ditanya : r ?

Dijawab :

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

31,4 = [tex]\frac{25}{360}[/tex] x L

31,4 = [tex]\frac{5}{72}[/tex] x L

L = 452,16 cm²

L = πr²

452,16 = 3,14 x r²

r² = 144

r = 12 cm


c. Diketahui :

r = 90 cm

Luas juring = 8.478 cm²

π = 3,14

Ditanya : [tex]\theta[/tex] = ?

Dijawab :

L = πr²

L = 3,14 x 90 x 90

L = 25.434 cm²

Luas Juring = [tex]\frac{\theta}{360}[/tex]L

8.478 = [tex]\frac{\theta}{360}[/tex] x 25.434

[tex]\frac{\theta}{360}[/tex] = [tex]\frac{8.478}{25.434}[/tex]

[tex]\theta[/tex] = 120°

18. bagian dari ruang adalah . pelajaran kelas 7 smp kurikulum 2013


Setiap ruang dipermukaan bumi memiliki karateristik atau ciri khas tertentu. Karateristik inilah yang kemudian menciptakan keterkaitan antar ruang dipermukaan bumi. Contoh dari keterkaitan antar ruang tersebut misalnya :

1. Peristiwa  banjir di Jakarta terjadi karena kerusakan hutan di daerah Bogor. Air hujan yang jatuh di daerah Bogor sebagian besar masuk ke sungai. Hanya sebagian kecil air hujan yang terserap oleh  tanah di Bogor. Akibatnya, Jakarta terkena  banjir yang airnya sebagian berasal dari wilayah Bogor.

2. Penduduk kota menghasilkan berbagai produk industri, seperti pakaian, kendaraan, barang-barang elektronik, dan lain-lain. Penduduk desa tidak menghasilkan produk-produk tersebut sehingga mereka pergi ke  kota untuk memperoleh barang-barang tersebut. Sebaliknya, penduduk  kota tidak menghasilkan bahan  pangan sehingga mereka memperolehnya dari penduduk  desa. Akibatnya, ada aliran barang dari  kota ke  desa dan aliran bahan makanan dari desa ke  kota.

3. Lapangan pekerjaan banyak tersedia di  kota, sedangkan di  desa hanya terbatas pada sektor pertanian. Akibatnya, banyak penduduk desa yang bepergian ke kota untuk bekerja atau mencari pekerjaan.
