2. ada yang tau rumus cepat untuk deret aritmatika


Un = a + (n-1) b
ket: a = suku pertama
b = bedanya
Un = urutan suku ke-n
n = suku ke-n[tex]$\x$Rumus mencari suku ke n :\\\\\boxed{$U_n$ = a + ( n - 1 ) b$}\\\\Rumus mencari jumlah n suku pertama :\\\\\boxed{$S_n$ = $\frac{n}{2}$ ( a + U_n$ )$} atau \boxed{$S_n$ = $\frac{n}{2}$ ( 2a + ( n - 1 ) b )$}[/tex]

3. suku ke-n suatu deret aritmatika adalah Un=3n-5. rumus jumlah n suku yang pertama deret tersebut adalah.. #butuh cepat buat besok


[tex]Un = a+(n-1)b = a + nb - b = nb + (a-b) = 3n - 5[/tex]
b = 3
a- b = -5, a - 3 = -5, a = -2

[tex]Sn= \frac{1}{2}n(a+Un)= \frac{1}{2}n((-2)+(3n-5)) = \frac{1}{2}n(3n-7) [/tex]

4. tentukan rumus suku ke -n pada tiap deret aritmatika di bawah ini, jika diketahui : a. Sn = 2n²+7n b. Sn = 1-2n²tolong cepat dijawab, please


Sn diganti dg 2n²+7n
S(n-1) diganti 2(n-1)²+7(n-1)

Un = Sn - S(n-1)
Un = [2n²+7n] - [2(n-1)²+7(n-1)]



Sn diganti 1 - 2n²
S(n-1) diganti 1 - 2(n-1)²
Un = Sn - S(n-1)
Un= [1-2n²] - [1-2(n-1)²]

Sn = n/2 (10 + 8n - 8)
Sn = n/2 (8n + 2)
