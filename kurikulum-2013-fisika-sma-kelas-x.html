9. kunci jawaban ipa fisika kelas 7 hal 152 kurikulum 2013​


Kabel Listrik dipasang agak kendur agar kabel tidak putus saat suhu turun dan terjadi penyusutan.Koefisien muai panjang besi sebesar 12 x 10⁻⁵/°C atau  1,2 x 10⁻⁶/°CPanjang Baja pada suhu 70°C adalah 40,04 cm

Pembahasan

Pemuaian

Saat benda dikenai kalor, maka akan terjadi pemuaian pada benda tersebut yang akan mengakibatkan pertambahan ukuran pada benda. dan sebaliknya saat kalor dilepaskan dari benda, akan terjadi penyusutan, besarnya pemuaian atau penyusutan tergantung pada koefisien muai benda dan perubahan suhu.

Pada Pemuaian Panjang berlaku persamaan :ΔL = Lo x α x Δt  Lt = Lo + ΔL  

atau  

Lt = Lo + (Lo x α x Δt)  

Keterangan :  

Lo = Panjang awal = m  

Lt = Panjang akhir = m  

ΔL = Pertambahan Panjang = m  

α = Koefisien muai panjang

Pada Pemuaian Luas berlaku persamaan :ΔA = Ao x β x Δt  

dan  

At = Ao + ΔA  

Atau  

At = Ao + (Ao x β x Δt)  

Keterangan :  

ΔA = Pertambahan Luas  

Ao = Luas awal  

At = Luas Akhir  

β = Koefisien Muai Luas  

β = 2 α  

Pada Pemuaian Volume berlaku persamaan :ΔV = Vo x γ x Δt  

dan untuk Volume akhir adalah :  

Vt = Vo + ΔV  

atau  

Vt = V0 + (V0 x γ x Δt)  

Keterangan  

ΔV = Pertambahan Volume  

γ = Koefisien Muai Ruang atau Volume  

besarnya γ = 3α  

Δt= Perubahan Suhu  

Vo= Volume awal  

Vt = Volume akhir  

Pengerjaan soal :

Soal Np 1 :

Pemasangan Kabel Listrik dibuat kendur agar saatterjadi penyusutan dimalam hari kabel tidak putus, seperti sudah dijelaskan diatas jika benda mendapatkan kalor akan mengalami pemuaian, dan jika benda melepaskan kalor (mengalami penurunan suhu) akan terjadi penyuutan. Perubahan panjang tersebut dirumuskan dalam persamaan :

ΔL = Lo x α x Δt

Keterangan :  

Lo = Panjang awal = m  

ΔL = Pertambahan Panjang = m  

α = Koefisien muai panjang =  

Soal No 2 :Diketahui :

t1 = Suhu Awal = 0°C

t2 = Suhu akhir = 100°C

Lo = Panjang awal besi = 200 cm

Lt = Panjang akhir besi = 200,24 cm

ΔL = Perubahan panjang besi = 200,24 cm - 200 cm = 0,24 cm

Ditanyakan :

α = Kofisien muai panjang besi = ...

Penyelesaian :

ΔL = Lo x α x Δt

0,24 = 200 x α x 100

0,24 = 2.000α

α = 0,24 : 2.000

α = (24 x 10⁻²) : (2 x 10³)

α = 12 x 10⁻⁵   = 1,2 x 10⁻⁶

Soal No 3 :Diketahui :

t1 = Suhu awal = 20°C

t2 = Suhu akhir = 70°C

Δt = Perubahan Suhu = 70 - 20 = 50°C

Lo = Panjang Baja mula - mula = 40 cm

α   = Koefisien muai baja = 0,00002/°C = 2 x 10⁻⁵/°C

Ditanyakan :

Lt = Panjang akhir Baja = ...

Penyelesaian :

Lt = Lo + (Lo x α x t)

Lt = 40 + (40 x 2 x 10⁻⁵ x 50)

Lt = 40 + (4000 x 10⁻⁵)

Lt = 40 x (4 x 10⁻²)

Lt = 40 + 0,04

Lt = 40,04 cm

Kesimpulan :

Kabel di pasang agak kendur agar tidak putus saat penyusutanKoefisien muai panjang besi adalah 12 x 10⁻⁵/°C atau 1,2 x 10⁻⁶/°CPanjang akhir baja adalah 40,02 cm

38. fisika kelas x kurikulum merdeka​


Luas plat besi tersebut berdasarkan angka penting adalah 50 cm².(Pilihan E). Soal tersebut merupakan soal tentang jangka sorong.

Penjelasan dengan langkah-langkah

Soal tersebut merupakan soal tentang pengukuran panjang. Panjang merupakan suatu ukuran yang memiliki satuan.

Satuan Panjang

km

  hm

      dam

            m

              dm

                 cm

                    mm

Setiap turun satu satuan, maka harus dikalikan dengan 10.

Jangka Sorong

Jangka sorong merupakan alat yang digunakan untuk mengukur panjang suatu benda dengan ketelitian 0,01 cm atau 0,1 mm. Jangka sorong dapat digunakan untuk mengukur ketebalan suatu benda, diameter luar, dan diameter dalam. Jangka sorong dapat juga digunkana untuk mengukur kedalaman suatu tabung.

Skala pada jangka sorong

Jangka sorong memiliki 2 skala yaitu skala utama dengan satuan ukuran cm dan skala nonius degan satuan ukuran mm.

Untuk menyelesaikan soal tersebut kita harus mengukur masing-masing panjang dan lebar dari plat besi, lalu kita baru bisa menentukan luas dari plat besi tersebut.

Penyelesaian soal

Diketahui:

Hasil pengukuran panjang plat besi adalah 11,33 cmHasil pengukuran lebar plat besi adalah 4,45 cm

Ditanyakan:

Berapa luas plat besi tersebut?

Jawab:

L = p x lL = 11,33 cm x 4,45 cmL = 50,4185 cm² ≈ 50 cm²

Jadi, luas plat besi tersebut adalah 50 cm² (pilihan E).
