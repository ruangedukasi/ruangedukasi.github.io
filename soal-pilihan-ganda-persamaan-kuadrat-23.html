9. Soal Pilihan Ganda Nilai yang memenuhi persamaan 2Log x = 3 adalah ....​


Logaritma

Kelas X SMA

[tex]\sf ²log \: \: x = 3 \\ \: \: \: \: \: \: \: \: \: \: \: \: \sf \: x= 2 {}^{3} \\ \: \: \: \: \: \: \: \: \: \: \sf \: x=8[/tex]

Nilai x dari persamaan ²log x = 3 adalah x = 8

PENDAHULUAN

Pengertian Logaritma

Logaritma ialah operasi invers yg merupakan kebalikan dari perpangkatan dengan penulisan a log b = c <—> a² = c.

ket:

a => basis atau bisa disebut dgn bilangan pokok.

b => numerus

c => hasil dari logaritma

PEMBAHASAN

²log x = 3

x = 2³

x = 2×2×2

x = 8

Kesimpulan:

Jadi, nilai x adalah 8

30. *Ini soalnya bukan pilihan ganda​


Jawaban:

a. (f o g)(x) = 6x² + 3

b. (g o f)(x) = 12x² + 12x + 4

c. (f o f)(x) = 4x + 3

d. (f o g)(4) = 99

Penjelasan dengan langkah-langkah:

a. (f o g)(x) = f(3x² + 1)

= 2(3x² + 1) + 1

= 6x² + 2 + 1

(f o g)(x) = 6x² + 3

b. (g o f)(x) = g(2x + 1)

= 3(2x + 1)² + 1

= 3(4x² + 4x + 1) + 1

= 12x² + 12x + 3 + 1

(g o f)(x) = 12x² + 12x + 4

c. (f o f)(x) = f(2x + 1)

= 2(2x + 1) + 1

= 4x + 2 + 1

(f o f)(x) = 4x + 3

d. (f o g)(x) = 6x² + 3

(f o g)(4) = 6(4)² + 3

(f o g)(4) = 6(16) + 3

(f o g)(4) = 96 + 3

(f o g)(4) = 99
