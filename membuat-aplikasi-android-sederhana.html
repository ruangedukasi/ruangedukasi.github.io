5. Bagaimana cara membuat aplikasi pada android? ​


Pake Android Studio

Caranya;

1. Create new project

- Buka android studio dan pilih start a new android studio project

- Isikan nama aplikasinya dan pilih lokasi penyimpanan project android studionya.

- Pilih minimum OS Android yang dibutuhkan untuk menjalankan aplikasimu

- Pilih empty activity

- Beri nama activity yang akan dibuat lalu klik Finish

2. Tampilan Environment Android Studio

- Kamu cukup focus pada main activity dan activity_main.xml

3. Melihat Tampilan Android

- Buka file activity_main.xml

4. Jalankan aplikasi android

- Klik tombol play berwarna hijau

- Pilih dimana kamu akan menjalankan aplikasi androidnya

- Untuk yang punya smartphone android, pergi ke developer options di menu settings android-nya → Lalu aktifkan opsi USB Debugging → Tancapkan smartphone ke komputermu via kabel USB → Jika muncul peringatan di android mengenai perizinan USB Debugging, klik saja ALLOW → Install driver smartphone android di komputermu → (Kembali ke Android Studio) → Maka device android-mu akan muncul di dialog tadi di bagian Connected Devices. Klik device android-mu lalu klik OK, tunggu hingga proses compiling selesai dan aplikasi android-mu akan berjalan di smartphone-mu.

5. Selesai

Android Studio sangat user-friendly. Dengan kesungguhan dan mau belajar kamu pasti bisa menguasainya dengan baik meskipun tidak mahir coding.

Install Aplikasi JDK

Anda harus terlebih dahulu mengInstall dan mensetting JDK (Java development kit)JDK adalah Software yang dipakai untuk kompilasi dari bahasa pemrograman Java ke bahasa pemrograman bytecode.

Install Aplikasi Android studio

Android Studio merupakan aplikasi Integrated Development Environment (IDE) untuk pembuatan aplikasi Android, menurut IntelliJ IDEA . Aplikasi Android Studio selain editor kode IntelliJ dan developer tools, Android Studio juga memberikan fitur lebih banyak dalam membuat aplikasi Android

Buat Interface dan Struktur Project

Jika Anda baru pertama kali membuka IDE Android Studio maka yang pertama yang harus Anda lakukan adalah membuat aplikasi pertama, yaitu dengan cara klik Start a new Android Studio project.

Membuat Linear layout

Linear Layout merupakan kelompok tampilan yang membuat tampilan sejajar dalam satu arah, dalam bentuk vertikal maupun horizontal. Dengan linear layout ini Anda dapat menetapkan arah layout mau ke arah horizontal maupun vertikal sesuai keinginan Anda.

Membuat Relative layout

Relative layout adalah salah satu tampilan group struktur code yang bisa mengatur komponen-komponen didalamnya atau yang dibungkus secara relative.

39. jelaskan cara membuat sebuah aplikasi android?​


Cara membuat aplikasi Android (Hello World Project)

1. Install Java JDK, Android SDK, dan Android Studio

2. Buka Android Studio

3. Buka File -> New -> New Project... untuk membuat project Android baru

4. Pilih Empty Activity dan tekan Next

5. Namain aplikasi tersebut, bahasa pemrograman, API Android dan pilih direktori penyimpanan lalu tekan Finish

6. Tunggu Gradle build disync terlebih dahulu dan akan terbuka tab activity_main.xml dan MainActivity.java secara dasar, file xml tersebut adalah tampilan depan aplikasi kita dan .java adalah code behind halaman tersebut

7. Kita buka file .xml tersebut dan lihat pada tampilan workspace dimana sudah ada Palette yang berisikan tool yang dapat digunakan secara drag and drop pada activity tersebut. Namun kita tidak akan menggunakan Design maka kita pindah tab ke Text

8. Jika dilihat pada file xml sudah ada serangkaian class yang digunakan untuk tampilan activity tersebut ada class android.support.constraint.ConstraintLayout dan TextView

9. Class ConstraintLayout digunakan untuk menentukan layout tampilan yang di dalam class tersebut apakah secara linear ke kanan relatif atau kaku mengikuti parent class. TextView adalah class yang menampilkan Text pada activity

10. Kita ganti ConstraintLayout menjadi RelativeLayout agar konten dalam class tersebut menjadi relatif, ketika diganti tulisan "Hello World" menjadi ke pojok kanan atas maka kita harus ganti attribut yang di dalam class TextView

11. Di dalam TextView ada attributes:

       app:layout_constraintBottom_toBottomOf="parent"

       app:layout_constraintLeft_toLeftOf="parent"

       app:layout_constraintRight_toRightOf="parent"

       app:layout_constraintTop_toTopOf="parent"

agar TextView menjadi ke tengah maka kita hapus layout_constraint menjadi:          

       android:layout_centerHorizontal="true"

       android:layout_centerVertical="true"

Kita juga perbesar tulisan dengan menaruh attribute dan id untuk identifikasi

       android:textSize="30dp"

       android:id="@+id/tvHelloWorld"

12. Kemudian tambahkan class EditText dan Button dengan attribute yang ada

13. Berikut adalah xml sementara dan tampilan: terlampir pada gambar

14. Lalu kita buka MainActivity.java dan dapat kita lihat sudah ada class dan import package, dan method onCreate, method tersebut adalah method yang berjalan pertama kali dalam life cycle Activity ini

15. Sekarang kita panggil Button, EditText, dan TextView tadi dengan cara menuliskan ini dibawah class MainActivity.java

   EditText etTulisan;

   TextView tvHelloWorld;

   Button btnUbah;

16. Lalu kita panggil mereka di method onCreate (Terlampir) dan panggil method Button yang bernama .setOnClickListener dimana jika diklik maka lakukan yang didalam method ini

17. Isi method tersebut dengan logic dimana jika onClick jalan maka tulisan yang sudah kita ketikan pada etTulisan dilempar ke tvHelloWorld

   protected void onCreate(Bundle savedInstanceState) {

       super.onCreate(savedInstanceState);

       setContentView(R.layout.activity_main);

       etTulisan = findViewById(R.id.etTulisan);
