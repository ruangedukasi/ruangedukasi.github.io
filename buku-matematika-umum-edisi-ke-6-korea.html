15. bahasa koreanya aku senang membaca buku ?


(나는) 책을 읽는 것을 좋아해.

( Aku suka membaca buku ).

Yang ada didalam kurung bisa dihilangkan, artinya tetap sama.Diakhir kalimat, bisa ditambahkan partikel 요 supaya terkesan formal.

Pembahasan

Pola kalimat bahasa Korea berbeda dengan kalimat Indonesia yang umumnya berpola S - P - O namun didalam bahasa Korea polanya S - O - P.

Arti per kata :

나 (na) artinya Aku.는 (neun) adalah Partikel topik.책 (chaeg) artinya Buku.을 (eul) adalah Partikel object.읽는 것을 (ilg-neun geos-eul) artinya Membaca (plural).좋아해 (joh-ahae) artinya Suka.

Penjelasan partikel :

는 (neun) disebut partikel topik karena menyertai hal yang diperbincangkan subject dipakai jikalau subjectnya berakhir vokal.

을 (eul) disebut partikel objek dipakai jikalau object atau frase bendanya berakhir konsonan.

Perlu diketahui :

• 좋아해 (joh-ahae) adalah kata kerja informal yang bentuk kamusnya 좋아하다 (joh-ahada) berubah sesuai aturan ketiga yaitu bilamana kata kerja berakhiran 하다 (hada) maka diganti 해요 (haeyo) menyesuaikan lawan bicara jika teman bisa hilangkan 요 (yo) menjadi 좋아해 (joh-ahae) atau 좋아해요 (joh-ahaeyo).

PERSAMAAN LINGKARAN

Bentuk umum persamaan lingkaran :

(x -a)² + (y -b)² = r²

untuk (1, -1) :

(1 -a)² + (-1 -b)² = r²

a² + b² -2a + 2b + 2 = r² ..... (1)

untuk (10, 2) :

(10 -a)² + (2 -b)² = r²

a² + b² -20a -4b + 104 = r² ..... (2)

untuk (6, -6) :

(6 -a²) + (-6 -b)² = r²

a² + b² -12a + 12b + 72 = r² ..... (3)

eliminasi (1) dan (2) :

a² + b² -20a -4b + 104 = r²

a² + b² -2a + 2b + 2 = r²

_____________________-

-18a -6b + 102 = 0

3a + b -17 = 0 ..... (4)

eliminasi (1) dan (3) :

a² + b² -2a + 2b + 2 = r²

a² + b² -12a + 12b + 72 = r²

____________________-

10a -10b -70 = 0

a -b -7 = 0 ..... (5)

eliminasi (4) dan (5) :

3a + b -17 = 0

a -b -7 = 0

___________+

substitusi a ke (5) :

a -b -7 = 0

(1 -6)² + (-1 + 1)² = r²
