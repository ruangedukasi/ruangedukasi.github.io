6. soal kemololan dan fraksi Mol pt3​


Penjelasan:

3. fraksi mol => X menyatakan perbandingan jumlah zat terlarut dengan jumlah zat total pelarut + terlarut

x1 = n1 / ( n1 + n2 )

x2 = n2 / ( n1 + n2 )

x1 + x2 = 1

x1 = fraksi mol terlarut

n1 = mol zat terlarut

x2 = fraksi mol pelarut

n2 = mol zat pelarut

4. a. Mr NaOH = Ar Na + Ar O + Ar H

=> 23 + 16 + 1

=> 40

mol NaOH = 20 / 40 => 0,5 mol

Mr H2O = ( 2x Ar H ) + Ar O

=> ( 2 x 1 ) + 16

=> 18

massa H2O = 36 ml x 1 gram / ml = 36 gram

mol H2O = 36 / 18 => 2 mol

b. fraksi mol NaOH: X2 = n2 / ( n1 + n2 )

X2 = 0,5 / ( 2 + 0,5 )

X2 = 0,5 / 2,5

X2 = 0,2

18. Cabang geografi yang secara khusus menyelidiki area tertentu adalah…


Jawaban:

Cabang geografi yang secara khusus menyelidiki area tertentu adalah Geografi Regional. Geografi regional merupakan salah satu cabang ilmu geografi. Geografi regional mengkaji suatu wilayah secara mendalam. Contohnya adalah Geografi Regional Asia yang mengkaji mengenai kondisi geografis benua Asia secara lengkap.

Penjelasan:

Geografi merupakan ilmu yang mempelajari mengenai persamaan dan perbedaan dari berbagai fenomena atau gejala geosfer yang terjadi di permukaan bumi. Dalam kajiannya geografi memerlukan beberapa ilmu penunjang. Ilmu penunjang kajian geografi diantaranya adalah hidrologi, geologi, biogeografi, dan lain sebagainya.
