13. tolong bantu penjabarannya gaes


Misal, x = 2k dan y = (2k + 1)

Maka,

(x + y)(x - y) = (2k + 2k + 1)(2k - (2k - 1))
= (4k + 1)(1)
= 4k + 1 (ganjil)

6^2(3x - y) = 6^2(3(2k) - (2k + 1))
= 6^2(4k + 1)
= 6^2.4k + 6^2 (genap)

(x + y^2)^2 - 1 = (2k + (2k + 1)^2)^2 - 1
= (2k + 4k^2 + 4k + 1)^2 - 1
= (4k^2 + 6k + 1)^2 - 1^2
= (4k^2 + 6k + 1 + 1)(4k^2 + 6k + 1 - 1)
= (4k^2 + 6k + 2)(4k^2 + 6k) (genap)

x^2 + 2xy + y + 3 = (2k)^2 + 2(2k)(2k + 1) + (2k + 1) + 3
= 4k^2 + 8k^2 + 4k + 2k + 4 (genap)

(x - 3)^2 + (y + 4)^2 = (2k - 3)^2 + (2k + 1 + 4)^2
= (4k^2 - 12k + 9) + (4k^2 + 20k + 25)
= 8k^2 + 8k + 34 (genap)

Jadi, yang ganjil adalah opsi A.
