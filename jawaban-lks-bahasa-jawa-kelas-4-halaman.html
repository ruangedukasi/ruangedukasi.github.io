11. kunci jawaban lks bhs jawa kelas 4 semester 2 halaman 42tolong di jawab mau dikumpulkan​


WangsulanNgrasakake saka tembung lingga rasa lan ng- + -ake.Nggoleki saka tembung lingga golek lan ng- + -i.Ngumbulake saka tembung lingga umbul lan ng- + -ake.Nyenengake saka tembung lingga seneng lan ny- + -ake.Ngresiki saka tembung lingga resik lan ng- + -i.Nggatekake saka tembung lingga gatek lan ng- + -ake.Nyangoni saka tembung lingga sangu lan ny- + -i.Ngunekake saka tembung lingga uni lan ng- + -ake.Ditaleni saka tembung lingga tali lan di- + -i.Satekane saka tembung lingga tekan lan sa- + -e.

Penjelasan

a. Tembung Lingga

Tembung lingga iku tembung kang durung owah saka asale. Tegese durung owah iku durung diwenehi ater-ater, seselan, panambang, lan pangrangkep.

Tuladha : tulis, dolan, jaluk, gambar,btakon, turu, pacul, adus, ukur, padhang, lsp.

b. Tembung Andhahan

Tembung andhahan iku tembung kang wis owah saka asale utawa linggane. Owahe bisa amarga diwenehi ater-ater, seselan, panambang, utawa dirangkep. Tembung lingga kang wis diowahi iku bakal malih dadi tembung andhahan.

Tuladha :

1) madhangi = m + pandhang + i

2) pitakon = pi + takon + an

3) dolanan = dolan + an

4) dijaluk = di + jaluk

5) ngadusi = ng + adus + i

6) nggoleki = ng + golek + i

7) tinulis = tulis + in

8) gegambaran = ge + gambar + an

9) macul = m + pacul

10) ngukur = ng + ukur

11) sapokna = sapu + na

12) talenana = tali + ana
