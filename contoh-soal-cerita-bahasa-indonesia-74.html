21. [74÷2]soal terlampir​


~Lingkaran

____________

DiketahuiPanjang Garis Singgung Persekutuan Dalam ( GSPD ) = 24 cmJarak dua titik pusat lingkaran ( p ) = 25 cmJari-jari lingkaran 1 ( r ) = 3 cm

Ditanya

Panjang jari-jari lingkaran yang lain ( R )

» Penyelesaian

GSPD = √[p² - (R + r)²]

GSPD² = p² - (R + 3)²

24² = 25² - (R + 3)²

576 = 625 - (R + 3)²

(3 + R)² = 625 - 576

(3 + R)² = 49

3 + R = √49

3 + R = 7

R = 7 - 3

R = 4 cm

• Kesimpulan

Jadi, Panjang jari-jari lingkaran yang lain adalah 4 cm

29. Quiz #74Soal:1/8 × 1/3​


[tex]Hasil \ dari \ \frac{1}{8} \times \frac{1}{3} \ adalah \ \frac{1}{24}[/tex].

Pembahasan

Pecahan adalah suatu bilangan dalam matematika yang dinyatakan dengan pembilang dan penyebut atau bisa juga diartikan sebagai perbandingan antara pembilang dan penyebut. Pecahan memiliki banyak jenis diantaranya sebagai berikut.

Pecahan biasa :

Pecahan biasa adalah pecahan yang nilai pembilangnya lebih kecil daripada nilai penyebutnya. Pecahan biasa misalnya pecahan [tex]\frac{1}{2}[/tex], [tex]\frac{2}{11}[/tex], dan lain sebagainya. Pecahan biasa dapat dituliskan sebagai berikut :

[tex]\boxed{\bold{ \frac{x}{y} }} \ \ \ \ x = pembilang \ \ \ \ y = penyebut[/tex]

Pecahan campuran :

Pecahan campuran adalah pecahan dalam wujud campuran, dalam arti terdiri dari bilangan bulat yang kemudian diikuti oleh pembilang dan penyebut. Pecahan campuran misalnya pada pecahan [tex]1\frac{2}{31}[/tex] dan [tex]2\frac{1}{5}[/tex]. Pecahan campuran dituliskan sebagai berikut :

[tex]\boxed{\bold{ a\frac{x}{y} }} \ \ \ \ a = bilangan \ bulat \ \ \ \ x = pembilang \ \ \ \ y = penyebut[/tex]

Pecahan desimal :

Pecahan desimal adalah pecahan yang dituliskan dalam bentuk kelipatan persepuluh, perseratus, perseribu, dan seterusnya. Pecahan desimal ditandai dengan adanya tanda koma (,). Satu angka di belakang koma berarti sama dengan persepuluh, dua angka dibelakang koma sama dengan perseratus, begitu pula seterusnya.Pecahan desimal misalnya 0,18, 2,11, dan lain sebagainya.

Pecahan persen :

Pecahan persen adalah pecahan yang bernilai persen atau perseratus. Pecahan persen ditandai dengan adanya tanda persen (%). Pecahan persen misalnya 297%, 12%, dan lain sebagainya.

Pecahan permil :

Pecahan permil adalah pecahan yang bernilai perseribu. Pecahan permil ditandai dengan adanya tanda permil (‰). Pecahan permil misalnya 33‰, 135‰, dan lain sebagainya.

Perbedaan antara "it" dan "them"

Untuk " it " , termasuk ke dalam golongan kata subject tunggal. Umumnya penggunaan kata " it " seperti menunjukkan sesuatu (itu) yang jumlahnya hanya satu (tunggal).

Sedangkan, untuk " them " (mereka) bukanlah jenis subject, melainkan object dengan bentuk jamak (banyak).

___________

• Pengertian Subject

Merupakan kata utama dalam memulai pembicaraan atau dialog.

• Pengertian Object

Merupakan kata yang umumnya digunakan atau di letakkan pada pola tempat akhir atau setelah dari predikat.

Misalnya seperti grammar, kata subject selalu diletakkan pada kata depan (utama). Berbeda dengan object yang diletakkan di akhir. Example :

Simple past tense :

(+) S + V2 + -ed/-d + O.(-) S + did + not + V1 + O.(?) Did + S + V1 + O + ?

Simple present tense :

(+) S + V1 + -es/-s + O.(-) S + does + V1 + O.(?) Does + S + V1 + O + ?

Present continuous tense :

(+) S + am/is/are + V1 + ing + O.(-) S + am/is/are + not + V1 + ing + O.(?) Am/is/are + S + V1 + ing + O + ?

Dan masih banyak lagi yang menggunakan penempatan seperti itu lainnya antara subject dan object.
