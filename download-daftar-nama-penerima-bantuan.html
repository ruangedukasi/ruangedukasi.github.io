2. Jelaskan langkah-langkah menyimpan (Download) gambar dari internet, dengan nama file Gambar_1 yang disimpan pada Data_D : Folder Download Gambar. Tolong bantuannya ​


Berikut adalah langkah-langkah untuk menyimpan (download) gambar dari internet dengan nama file "Gambar_1" dan menyimpannya dalam folder "Download Gambar" di drive D, yaitu:

Buka browser web dan kunjungi situs web tempat gambar yang ingin Anda download.Temukan gambar yang ingin Anda download dan pastikan Anda memiliki izin untuk mengunduhnya.Klik kanan pada gambar dan pilih opsi "Save Image As" (Simpan Gambar Sebagai) atau serupa, tergantung pada browser yang Anda gunakan.Sebuah jendela dialog akan muncul, memungkinkan Anda untuk memilih lokasi penyimpanan gambar.Arahkan jendela dialog ke drive D: dengan mengklik ikon drive di bagian atas jendela dan memilih "D:/".Buka folder "D:" dan buat folder baru dengan nama "Download Gambar" jika folder tersebut belum ada.Masuk ke folder "Download Gambar".Di jendela dialog, berikan nama file "Gambar_1" (tanpa tanda kutip) pada kolom "Nama file" atau "File name".Pastikan format file yang dipilih sesuai dengan format gambar yang ingin Anda download (misalnya, .jpg, .png, .gif).Klik tombol "Save" (Simpan) atau serupa di jendela dialog. Gambar akan mulai diunduh dan disimpan di folder "Download Gambar" di drive D: dengan nama file "Gambar_1".

Pembahasan

Download adalah proses pengambilan data atau file dari internet atau sumber lainnya dan menyimpannya ke perangkat lokal, seperti komputer, ponsel, atau tablet. Aktivitas download umumnya dilakukan untuk mendapatkan konten seperti gambar, video, musik, dokumen, perangkat lunak, dan berbagai jenis file lainnya.

Proses download melibatkan transfer data dari sumbernya ke perangkat pengguna melalui jaringan internet. Setelah file berhasil diunduh, file tersebut tersimpan dalam penyimpanan perangkat pengguna dan dapat diakses secara offline. Download sering dilakukan dengan menggunakan browser web, di mana pengguna dapat mengklik tautan atau gambar di situs web untuk memulai proses download.

Untuk mendownload gambar dari internet, Anda dapat menggunakan menu-menu atau opsi berikut, tergantung pada browser web yang Anda gunakan:

Google Chrome:Klik kanan pada gambar dan pilih "Save Image As" (Simpan Gambar Sebagai).Atau, klik kanan pada gambar dan pilih "Open image in new tab" (Buka gambar di tab baru). Kemudian, di tab baru, klik kanan pada gambar dan pilih "Save Image As" (Simpan Gambar Sebagai).Mozilla Firefox:Klik kanan pada gambar dan pilih "Save Image As" (Simpan Gambar Sebagai).Atau, klik kanan pada gambar dan pilih "View Image" (Lihat Gambar). Setelah gambar terbuka di tab baru, klik kanan pada gambar dan pilih "Save Image As" (Simpan Gambar Sebagai).Microsoft Edge:Klik kanan pada gambar dan pilih "Save picture as" (Simpan gambar sebagai).Atau, klik kanan pada gambar dan pilih "Open image in new tab" (Buka gambar di tab baru). Kemudian, di tab baru, klik kanan pada gambar dan pilih "Save picture as" (Simpan gambar sebagai).Safari:Klik kanan pada gambar dan pilih "Save Image As" (Simpan Gambar Sebagai).Atau, klik kanan pada gambar dan pilih "Open Image in New Tab" (Buka Gambar di Tab Baru). Kemudian, di tab baru, klik kanan pada gambar dan pilih "Save Image As" (Simpan Gambar Sebagai).

IDM ( Internet Download Manager )

Dapat membuka Kecepatan Lebih dari 32 Koneksi Download ( tergantung bandwith nya )

ialah sebuah perangkat lunak (software) yang digunakan untuk mengunduh data atau file yang terdapat di internet dan juga meneruskan kembali (resume). Program ini diproduksi di New York, Amerika ini menduduki posisi teratas dalam memaksimalkan kecepatan pengunduhan data (download). Tampilan antarmuka (interface) yang sederhana menjadikan software IDM lebih bersahabat dengan user atau pemakainya. Seringkali kita memanfaatkan IDM untuk mendownload video, aplikasi, dan file lainnya seperti dari facebook, google drive, atau yang lainnya. Untuk mendownloadnya bisa mengunjungi link internet download manager, atau idm gratis versi idm 7.1.

Flash Get

Berbasis mirip seperti torrent hanya saja aplikasi ini bisa menggunakan link download, tidak ada konkesi download

software download manager yang cukup ternama dan kehadirannya pun terbilang sudah cukup lama, serta memiliki pengguna yang cukup banyak. FlashGet menggunakan Multi-server Hyper-threading Transportation (MHT), dimana fitur ini sudah mendukung beragam protokol dan mengatur dokumen dengan baik. Bahkan dengan fitur tersebut, proses download bisa meningkat lebih cepat hingga 10x lipat.

Selain itu, FlashGet juga sudah mendukung beberapa protokol ternama seperti HTTP, FTP, Bittorrent, eMule dan masih banyak lagi.

uTorrent

Aplikasi ini sangat cocok untuk file ukuran besar, dan torrent tidak bisa akses atau download dari link normal,

merupakan sistem download file dengan menggunakan peer to peer. File torrent yang kita temukan selalu berextensi “*.torrent” dengan kapasitas yang hanya beberapa kilobit. Didalam file tersebut berisi informasi file yang akan di download, mulai dari kapasitas file, jumlah filenya, dan dimana file harus di download. Jadi file torrent ini bukan file yang sebenarnya, dan file yang kita donwload bukan berada di dedicate server tetapi dari berbagai jaringan komputer peer to peer yang saling terhubung dengan mengunakan aplikasi client torrent. Jadi setiap komputer yang mendonwload file tersebut otomatis menjadi server share buat yang lainnya. Intinya semua komputer saling berbagi antara satu dengan yang lain.

Download Accelerator Plus (DAP)

memiliki tampilan yang cukup mudah dimengerti. DAP mampu meningkatkan proses download dengan cepat, karena telah dilengkapi dengan fitur Automatic Mirroring Speed Boost. DAP juga memiliki kemampuan yang sama seperti FlashGet, mampu melakukan proses scanning melalui software antivirus yang digunakan, apakah file yang akan di download aman atau tidak.

Tersedia juga tombol Pause apabila koneksi Internet terputus. Bahkan, bila file yang akan di download rusak, tetap bisa dilanjutkan proses download-nya. DAP juga menyediakan sebuah layar preview apabila Anda hendak men-download sebuah file video

Free Download Manager (FDM) memiliki fitur dan tampilan yang tidak jauh berbeda dengan DAP. FDM merupakan software download manager gratis yang mampu meningkatkan kecepatan proses download berbagai file di Internet. Selain itu, FDM bisa dikatakan 100% aman karena berupa software open source yang didisitribusikan oleh lisensi GPL. Beberapa keunggulan FDM

merupakan sebuah software download manager gratis yang dirancang khusus untuk Web terbaru (2.0). Orbit Downloader juga mampu mencegah proses download file yang mengandung spyware, adware, atau malware. Selain itu, Orbit Downloader memiliki kemampuan download file video/audio dari website streaming ternama seperti YouTube, MySpace, Imeem, dan Pandora. Selain itu, software download manager ini juga memudahkan para penggunanya untuk men-download file dari RapidShare.

18. Sebuah perusahaan internet mengenakan biaya bulanan sebesar $ 50 dan $ 0,1 per menit saat men-download. Diterima tagihan untuk satu bulan sebesar $ 111. Hitung jumlah total menit pemakaian untuk download.(beserta penjelasannya)​


Jawab:

m = 610

Penjelasan dengan langkah-langkah:

Kita dapat memisalkan m sebagai jumlah  menit  download.

Kita tahu

Total tagihan = 111

Ini termasuk biaya bulanan dan biaya download.

Jadi,

biaya Bulanan + biaya download = total tagihan  ----- (1)

dimana,

biaya Bulanan = 50      

biaya download = (jumlah menit pemakaian) (biaya per menit) = (m) (0,1) = 0,1 m

Menggunakan persamaan (1) akan memberi kita:

50 + 0,1 M = 111

0,1 m = 61

m = 610

39. tolong bantu jawab kakkkkk1.) Paijo mendownload file 3,6 mb,ternyata data yang diterima Paijo sleesai dalam 1 menit berapa trou put nya?​


Jawaban:

Diketahui :

ukuran file = 3,6 mb = 3.600.000 bit

Waktu = 1 menit = 60 detik

Ditanya = throughput

Jawaban :

rumus Throughput = ukuran file / waktu per detik

= 3.600.000 bit / 60 detik

= 60.000 bit / detik

= 60 kilobit/detik

Jadi throughput Paijo adalah 60 kilobit/detik.
