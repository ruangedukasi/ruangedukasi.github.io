Penjelasan dengan langkah-langkah:

1. a. Volume Bangun = V Kerucut + V Tabung + V Bola

= (1/3 x phi x r x r x t) + (phi x r x r x t) + (1/2 x 4/3 x r x r x r)

= (1/3 x 3,14 x 5 x 5 x 12) + (3,14 x 5 x 5 x 12) + (2/3 x 5 x 5 x 5)

= 314 + 942 + 83,3

= 1.339,3 cm3.

Jadi Volume bangun tersebut adalah 1.339,3 cm3.

b. Volume Bangun = 2 x Volume Limas Segiempat + Volume Kubus

= (2 x 1/3 x s x s x t) + (p x l x t)

= (2/3 x 10 cm x 10 cm x 13 cm) + (10 cm x 10 cm x 12 cm)

= 866,7 cm3 + 1.200 cm3

= 2.066,7 cm3.

Jadi volume bangun adalah 2.066,7 cm3.

c. Volume Bangun = Volume Balok + Volume ½ Tabung

= (p x l x t) + (1/2 x phi x r x r x t)

= (20 x 10 x 8) + (1/2 x 3,14 x 5 x 5 x 20)

= 1.600 + 785

= 2.385 cm3.

Jadi volume bangun adalah 2.385 cm3.

2. Volume 3 Miniatur Rumah = 3 x (Volume Kubus + Volume Prisma)

= 3 x ((s x s x s) + (1/2 x a x t x t.prisma)

= 3 x ((5 cm x 5 cm x 5 xm) + (1/2 x 6 cm x 4 cm x 5 cm)

= 3 x (125 cm3 + 60 cm3)

= 3 x 185 cm3

= 555 cm3.

Jadi volume 3 miniatur rumah adalah 555 cm3.

3. Volume Mainan = (phi x r x r x t) + (phi x r x r x t) + (phi x r x r x t)

= (22/7 x 7 x 7 x 15) + (22/7 x 21/2 x 21/2 x 15) + (22/7 x 14 x 14 x 15)

= 2.310 + 5.197,5 + 9.240

= 16.747,5 cm3.

Jadi volume mainan kue tart adalah 16.747,5 cm3.

30. Siti mempunyai 2buku tulis jenis a,5buku tulis jenis b,dan 7buku tulis jenis c.perbandingan banyak halaman setiap buku tulis jenis a dan b adalah 6:7,perbandingan buku jenis a dan c adalah 3:4.jumlah halaman semua buku tulis tersebut 824. Banyak halaman setiap buku jenis b adalah


Misalkan:
- banyak halaman buku jenis a adalah A
- banyak halaman buku jenis b adalah B
- banyak halaman buku jenis c adalah C

Oleh karena perbandingan banyak halaman setiap buku tulis jenis a dan b adalah 6 : 7, maka

A : B = 6 : 7
A / B = 6 / 7 
A = 6/7 B ... (i)

Oleh karena perbandingan banyak halaman setiap buku jenis a dan c adalah 3 : 4, maka

A : C = 3 : 4
A / C = 3 / 4 
C = 4/3 A 
C = 4/3 x 6/7 B
C = 8/7 B ... (ii)

Oleh karena jumlah halaman semua buku tulis milik Siti adalah 824, sedangkan Siti memiliki 2 buku tulis jenis a, 5 buku tulis jenis b, dan 7 buku tulis jenis c, maka

824 = 2A + 5B + 7C
824 = (2 x 6/7 B) + 5B + (7 x 8/7 B)
824 = 12/7 B + 5B + 8B
824 = 103/7 B
B = 7/103 x 824
B = 7 x 8
B = 56

Jadi, banyak halaman dari buku jenis b adalah 56.

5. Ahmad dan Udin berdiri saling membelakangi untuk main tembak-tembakan pistol bambu. Ahmad berjalan 20 langkah ke depan kemudian 15 langkah ke kanan. Pada saat yang sama, Udin berjalan 16 langkah ke depan kemudian 12 langkah ke kanan. Udin berhenti kemudian menembak Ahmad.

Penyelesaian :

a.  Gambar dari situasi soal diatas yg menggunakan bidang kartesius ada pada lampiran.

b.  Jarak langkah mereka berdua saat Udin menembak Ahmad bisa menggunakan dua cara.

 I.  Menghitung petak langkah dengan menggunakan pythagoras

c² = a² + b²

AU² = (20 + 16)² + (15 + 12)

      = 36² + 27²

      = 1296 + 729

      = 2025

AU = √2025

AU = 45 satuan langkah

Jadi jarak mereka berdua saat Udin menembak Ahmad dengan pistol bambu adalh 45 langkah.

 II. Menghitung jarak dua titik antara Ahmad dan Udin yaitu (-20 , 15) dan (16 , -12)

Jarak AU = [tex]\sqrt{ (&#10;x_{2}-x_{1}) ^{2} + (y_{2} - y_{1}) ^{2}}[/tex]

               = [tex]\sqrt{ (16 - (-20))^{2} + (-12 - 15)^{2}}[/tex]

               = [tex]\sqrt{ (16 + 20)^{2} + (-12 - 15)^{2} }[/tex]

               = [tex]\sqrt{ 36^{2} + (-27)^{2} }[/tex]

               = [tex]\sqrt{1296 + 729}[/tex]

               = √2025

               = 45 langkah

Jadi jarak mereka berdua saat Udin menembak Ahmad dengan pistol bambu adalh 45 langkah.

 

6. Seorang atlet tenis mengajukkan pertanyaan kepada wasit. Suara atlet mampu didengar wasit hanya pada jarak maksimum 30 kaki. Jarak antara wasit dan atlet = 24 kaki . Ketinggian wasit melihat = 12 kaki . Tinggi atlet = 5 kaki.

Penyelesaian :

c² = a² + b²

x² = 24² + (12 - 5)²

   = 24² + 7²

   = 576 + 49

   = 625

x  = √625

x  = 25 kaki

8. Seorang penyelam dari tim SAR mengaitkan dirinya pada tali sepanjang 25 m untuk mencari sisa-sisa bangkai pesawat di dasar laut. Laut diselami memiliki kedalaman 20 meter dan dasarnya rata. Berapa luas daerah yang mampu dijangkau oleh penyelam tersebut?

Penyelesaian :

Panjang tali (s) = 25 m

Kedalaman (t) = 20 m

Sebelumnya kita cari dulu panjang jari-jari dengan menggunakan pythagoras.

r² = s² - t²

   = 25² - 20²

   = 625 - 400

   = 225

r = √225

r = 15 m

Luas daerah yang dijangkau

L lingkaran = π r²

                   = 3,14 × 15 × 15 m²

                   = 3,14 × 225 m²

                   = 706,5 m²

9. Tentukan panjang AG dari bangun berikut

Penyelesaian :

Panjang AG pada kubus dan balok merupakan diagonal ruang.

a. Kubus

AG² = HG² + FG² + BF²

AG² = 10² + 10² + 10²

AG² = 10² × 3

 AG = 10√3

Jadi panjang AG pada kubus adalah 10√3

b. Balok

AG² = 5² + 5² + 10²

AG² = 150

 AG = 5√6

10. Diameter Bola A dan bola B berturut-turut adalah 8 dan 18. Jika jarak ujung tali l dan n pada kawat adalah 5 dan panjang tali l adalah 10,berapakah panjang minimum tali n agar kedua tali bisa sejajar dan bola tidak saling menekan?

Penyelesaian :

Dari perpanjangan tali n, sehingga terbentuk persegi panjang ADEF dan segitiga ABD

panjang tali l = 10 cm

AF = DE = ¹/₂ × 8 = 4 cm

BC = ¹/₂ × 18 = 9 cm

AB = AF + BC = 4 + 9 = 13 cm

EF = AD = 5 cm

Kita lihat segitiga siku-siku ADB :

AB² = AD² + BD²

13² = 5² + BD²

169 = 25 + BD²

BD² = 169 - 25

BD² = 144

 BD = √144

 BD = 12 cm

CD = BD - BC

     = 12 - 9

     = 3 cm

Panjang tali n = panjang tali l + DE + CD

                      = 10 cm + 4 cm + 3 cm

                      = 17 cm

Jadi panjang minimum tali n agar kedua tali bisa sejajar dan bola tidak saling menekan adalah 17 cm
