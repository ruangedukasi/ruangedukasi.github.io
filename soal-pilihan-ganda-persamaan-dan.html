13. INI SOAL MATERI PELUANG BUKAN PILIHAN GANDA dua buah dadu dilempar bersamaan, tentukan peluang munculnya a. dadu pertama dan kedua angka ganjil b. jumlah mata dadu 8PAKAI CARANYA​


Penjelasan dengan langkah-langkah:

ruang sampel pelemparan 2 buah dadu = {

(1,1), (1,2), (1,3), (1,4), (1,5), (1,6),

(2,1), (2,2), (2,3), (2,4), (2,5), (2,6),

(3,1), ... dst sampe (6,6)}

banyak anggota ruang sampel = 6 × 6 = 36

himpunan kejadian munculnya dadu pertama dan kedua ganjil = {

(1,1), (1,3), (1,5),

(3,1), (3,3), (3,5),

(5,1), (5,3), (5,5)}

banyak kejadian = 9

himpunan kejadian munculnya jumlah mata dadu 8 = {

(2,6), (3,5), (4,4), (5,3), (6,2)}

banyak kejadian = 5

a. peluang munculnya dadu pertama dan kedua ganjil

= banyak kejadian munculnya dadu pertama dan kedua ganjil / banyak anggota ruang sampel

= 9/36

= 1/4

b. peluang munculnya jumlah mata dadu 8

= banyak kejadian munculnya jumlah mata dadu 8 / banyak anggota ruang sampel

= 5/36

27. contoh soal pilihan ganda persamaan dan pertidaksamaan nilai mutlak linear satu variabel


Contoh soal pilihan ganda persamaan dan pertidaksamaan nilai mutlak linear satu variabel

Pembahasan :

Definisi nilai mutlak :
|x| = x jika x ≥ 0
|x| = -x jika x < 0

|x| = √(x²)

Persamaan nilai mutlak (c > 0)
1) |ax + b| = c
=> ax + b = c atau ax + b = -c
2) |ax + b| = |cx + d|
=> (ax + b)² = (cx + d)²

Pertidaksamaan nilai mutlak (a> 0, c > 0)
1) |ax + b| < c
=> -c < ax + b < c
2) |ax + b| > c
=> ax + b < -c atau ax + b > c
3) |ax + b| < |cx + d|
=> (ax + b)² < (cx + d)²
