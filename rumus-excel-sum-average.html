8. 43. Untuk menjumlahkan data pada beberapa cell dalam program microsoft excel digunakan rumus ....A. AverageC. minimumB. MaximumD. sum​


Untuk menjumlahkan data pada beberapa cell dalam program Microsoft excel digunakan rumus D. Sum

Pembahasan

SESUAI OPSI SOAL:

A. Average

Adalah rumus dalam exvel yang berfungsi untuk menghitung jumlah nilai rata-rata dari beberapa cell dalam sebuah data maupun tabel yang tersaji di program excel, perumusan average adalah =AVERAGE(cell 1:cell 2).

B. Minimum

Atau juga disingkat min, yang dalam excel berfungsi untuk menghitung jumlah nilai yang paling rendah dari sebuah data dalam tabel di beberapa cell pada program excel, perumusan min adalah =MIN(cell 1:cell 2).

C. Maximum

Atau juga disingkat max, yang dalam excel berfungsi untuk menghitung jumlah nilai yang paling tinggi dari sebuah data dalam tabel di beberapa cell pada program excel, perumusan min adalah =MAX(cell 1:cell 2).

D. Sum

Adalah rumus dalam exvel yang berfungsi untuk menghitung jumlah nilai secara keseluruhan dari sebuah data dalam tabel di beberapa cell pada program excel, perumusan sum adalah =SUM(cell 1:cell 2).

27. Sebutkan penggunaan dari beberapa fungsi yang Ada pada Ms. Excel SUM AVERAGE MAX COUNTIF


Sum memiliki fungsi umum

=sum(Range)

Berguna untuk mencari total data pada range

Average memiliki fungsi umum

=average(range)

Berguna untuk mencari rerata dari data pada range

Max memiliki fungsi umum

=max(range)

Berguna untuk mencari nilai maksimal/nilai terbesar dari data pada range

Countif memiliki fungsi umum

=countif(range;"kriteria ")

Berguna untuk mencari banyak data dengan kriteria tertentu

Contoh:

=sum(A12: A23) → mencari total data pada sel A12 sampai sel A23

=average (B1:B5) → mencari rerata data pada sel B1 sampai sel B5

=max(A2: C5) → mencari nilai terbesar dari sel A2 sampai sel C5

=countif(A1: A21;"sehat") → mencari banyak data dari sel A1 sampai A21 yang berisi data " sehat "

34. cari rumus sum maxmintcount average​


Penjelasan:

Rumus-rumus statistik dasar yang sering digunakan adalah sebagai berikut:

1.Sum (Jumlah) - merupakan hasil penjumlahan dari semua data dalam sampel atau populasi.Rumus: [tex]Σx = x1 + x2 + ... + xn[/tex]Max (Nilai Maksimum) - merupakan nilai tertinggi dari semua data dalam sampel atau populasi.Rumus: Max(x) = nilai tertinggi dari x1, x2, ..., xnMin (Nilai Minimum) - merupakan nilai terendah dari semua data dalam sampel atau populasi.Rumus: Min(x) = nilai terendah dari x1, x2, ..., xnCount (Jumlah Data) - merupakan jumlah keseluruhan data dalam sampel atau populasi.Rumus: n = jumlah data dalam sampel atau populasiAverage/Mean (Rata-rata) - merupakan nilai rata-rata dari semua data dalam sampel atau populasi.Rumus: [tex]μ = (Σx) / n[/tex]

Dimana:

Σx adalah jumlah dari semua data dalam sampel atau populasi.

Σx adalah jumlah dari semua data dalam sampel atau populasi.n adalah jumlah data dalam sampel atau populasi.

Σx adalah jumlah dari semua data dalam sampel atau populasi.n adalah jumlah data dalam sampel atau populasi.μ adalah rata-rata atau mean.
