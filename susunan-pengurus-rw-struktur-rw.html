~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Diketahui :
Gambar Diagram Lingkaran Tentang Jumlah Peserta senam jantung sehat
RW 01 = 20 % → Peserta = _____?
RW 02 = ___? → Peserta = _____?
RW 03 = 19% →  Peserta = ___?
RW 04 = ___? → Peserta = 28 0rang
RW 05 = 22%  → Peserta = ___?

SOAL ini bisa dikerjakan JIKA ASUMSI presentase Peserta dari RW 02 SAMA dengan RW 04, maka :

%RW 01 + %RW 02 + %RW 03 + %RW 04 + %RW05 = 100%

20% + x + 19% + x + 22% = 100%
2x + 61% = 100%
2x = 100% - 61%
2x = 39%
x = 19,5%

Maka jumlah peserta senam dari RW 01 adalah 
= (20% / 19,5%) x 28 orang

= 28 0rang

35. Di suatu desa, jumlah penduduk RW 01 dan RW 02 adalah 6.800 jiwa. Setengah jumlah penduduk RW 01, 150 jiwa lebih banyak dari sepertiga penduduk RW 02. Banyak penduduk RW 01, adalah … jiwa.


Banyak penduduk RW 01 adalah 2.900 jiwa.

=======================================

Sistem persamaan linear dua variabel adalah sebuah sistem atau kesatuan dari beberapa persamaan linear dua variabel yang sejenis. Variabel dalam sistem persamaan ini merupakan peubah atau pengganti suatu nilai atau bilangan dan akan dicari penyelesaiannya.

PEMBAHASAN :

Perhatikan kembalu soal di atas.

• Di suatu desa, jumlah penduduk RW 01 dan RW 02 adalah 6.800 jiwa. Asumsikan jumlah penduduk RW 01 sebagai x dan jumlah penduduk RW 02 sebagai y. Maka, x + y = 6.800. Dan x = 6.800 - y.

• Setengah jumlah penduduk RW 01, 150 jiwa lebih banyak dari sepertiga penduduk RW 02. Berarti :

½ x = ⅓y + 150 (ingat, x = 6.800 - y)

½(6.800 - y) = ⅓y + 150

3.400 - y/2 = y/3 + 150 (kalikan 6)

20.400 - 3y = 2y + 900

5y = 19.500

y = jumlah penduduk RW 02 = 3.900 jiwa.

Substitusikan nilai y ke persamaan x + y = 6.800.

x + 3.900 = 6.800

x = 6.800 - 3.900

x = jumlah penduduk RW 01 = 2.900 jiwa.
