macro pada Excel adalah fitur yang memungkinkan pengguna untuk merekam dan memutar ulang serangkaian tindakan yang sering dilakukan dalam program Excel. Fitur ini pertama kali diperkenalkan pada tahun 1993 dalam Microsoft Excel versi 5.0 dan menjadi semakin populer sejak itu. Dengan adanya fitur macro, pengguna dapat menghemat waktu dan meningkatkan efisiensi dalam melakukan tindakan-tindakan yang sering dilakukan dalam program Excel. Saat ini, pengguna juga dapat mengakses dan mengedit kode makro secara langsung untuk membuat makro yang lebih kompleks dan disesuaikan dengan kebutuhan mereka.

10. Tipe dalam penyimpanan yang digunakan untuk menyimpan file yang terdapat macro didalamnya dan macro yang sudah dibuat akan tersimpan di dalam file, yaitu tipe…. Pilih salah satu: a. Excel Binary Workbook b. Excel Macro-Enable Workbook c. Excel Workbook d. Excel Add-In​


Jawaban:

b. Excel Macro-Enable Workbook

Penjelasan:

Berikut ini sedikit penjelasan dalam penggunaannya.

1. Excel Workbook, Menyimpan dengan tipe ini seperti menyimpan file excel atau workbook pada umumnya. Pada file ini tidak terdapat Macro.

2. Excel Macro-Enable Workbook, Menyimpan dengan tipe ini digunakan untuk menyimpan file yang terdapat macro didalamnya. Macro yang sudah dibuat akan tersimpan di dalam file.

3. Excel Binary Workbook, Menyimpan dengan tipe ini digunakan untuk menyimpan file yang terdapat macro didalamnya akan tetapi macro akan tersimpan pada system Microsoft Office Excel.

4. Excel Add-In, Menyimpan dengan tipe ini digunakan untuk menyimpan file macro yang dapat anda gunakan sebagai Add-in atau program tambahan dalam Program Excel.

Langkah-langka Cara menyimpan file Macro :

1. Lakukan Penyimpanan dengan tombol SAVE

2. Akan tampil peringatan “the following features cannot be saved in Macro-free workboks” peringatan ini menginformasikan bahwa file ini terdapat VB Project (macro). Jika anda pilih Yes maka Macro yang sudah dibuat akan dihapus, dikarenakan workbook biasa tidak diperkenenkan untuk menyimpan macro.

3. Pilih No Untuk menyimpan macro.

4. Lakukan penyimpanan didalam folder tujuan. Ubahlah Save as Type Menjadi Excel macro-enable Workbook (xlsm).

5. Klik Save

CATATAN: Karakter pertama dari nama makro harus berupa huruf. Karakter berikutnya bisa berupa karakter huruf, angka, atau garis bawah. Spasi tidak bisa digunakan dalam nama makro; karakter garis bawah berfungsi baik sebagai pemisah kata. Untuk menetapkan kombinasi Control (Ctrl) tombol pintasan untuk menjalankan makro, dalam kotak tombol pintasan , ketik huruf apa pun (baik huruf besar atau huruf kecil akan berfungsi) yang ingin Anda gunakan. Sebaiknya gunakan Ctrl + Shift (huruf besar) tombol kombinasi, karena tombol pintasan akan menimpa tombol pintasan Excel default yang sama saat buku kerja yang berisi makro terbuka. Sebagai contoh, jika Anda menggunakan Ctrl + z (Batalkan), Anda akan kehilangan kemampuan untuk Batalkan dalam contoh Excel.

Saat Anda memilih Buku kerja makro pribadi, Excel membuat buku kerja makro pribadi yang tersembunyi (Personal.xlsb) jika belum ada, dan menyimpan makro di buku kerja ini. Di Windows 10, Windows 7dan Windows Vista, buku kerja ini disimpan dalam C:\Users\user name\AppData\Local\Microsoft\Excel\XLStart folder. Di Microsoft Windows XP, buku kerja ini disimpan di C:\Documents and Settings\user name\Application Data\Microsoft\Excel\XLStart folder. Buku kerja di XLStart folder dibuka secara otomatis setiap kali Excel dimulai, dan kode apa pun yang sudah Anda simpan di buku kerja makro pribadi akan tercantum di dialog makro, yang dijelaskan di bagian berikutnya. Berikut adalah langkah-langkah untuk membuat gambar dalam aplikasi Microsoft Excel:

1. Buka Microsoft Excel.

2. Buka lembar kerja yang ingin Anda masukkan gambar.

3. Klik tab "Insert".

4. Di grup "Illustrations", klik tombol "Picture".

5. Pilih gambar yang ingin Anda masukkan dari komputer Anda.

1. Buka Microsoft Excel.

2. Buka lembar kerja yang ingin Anda buat gambar.

3. Klik tab "Insert".

4. Di grup "Illustrations", klik tombol "Drawing".

5. Pilih alat yang ingin Anda gunakan untuk membuat gambar.

6. Mulailah menggambar gambar Anda di dalam lembar kerja Excel.

Setelah Anda membuat gambar, Anda dapat mengubah ukuran, posisi, dan formatnya sesuai keinginan.

30. Bagaimana cara mengunakan macro di excel?jelaskan dengan lengkap!

Tempatkan Kursor pada E2, lalu masukkan formula : =C2*D2 Tekan Enter.Copy Cell E2 dengan Ctrl + CLalu Blok (sorot) E3 : E6, lalu paste dengan Ctrl + Vtempatkan kursor pada E7 lalu pilih fungsi Autosum  di menu Home. Tekan EnterKembali ke Menu Developer lalu klik Stop Recording. Sama seperti contoh pertama, anda dapat melihat kode yang dihasilkan bukalah VBE (Visual Basic Editor) dengan menekan Alt + F11
