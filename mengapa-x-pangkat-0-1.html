9. 2(25)pangkat x+1 + 5 pangkat x+2-3 = 0


Terdapat persamaan berikut : 2 x [tex] 25^{x + 1}[/tex] + [tex] 5^{x + 2}[/tex] - 3 = 0 . Hitunglah nilai x yg memenuhi persamaan berikut ! Jawaban Pendahuluan

Bentuk [tex] ap^{2x}[/tex] + [tex] bp^{x}[/tex] + c = 0 merupakan persamaan eksponen yg dapat dibuat menjadi bentuk persamaan kuadrat. Perhatikan persamaan berikut [tex] ap^{2x}[/tex] + [tex] bp^{x}[/tex] + c = 0 identik dengan [tex] a(p^{x})^{2}[/tex] + [tex] b(p^{x})[/tex] + c = 0. Jika dimisalkan y = [tex] p^{x}[/tex] maka persamaan [tex] ap^{2x}[/tex] + [tex] bp^{x}[/tex] + c = 0  menjadi ay² + by + c = 0. Selanjutnya selesaikan persamaan tersebut dengan cara mencari akar - akar persamaan kuadrat.  

Pembahasan

2 x [tex] 25^{x + 1}[/tex] + [tex] 5^{x + 2}[/tex] - 3 = 0

2 x [tex] 5^{2 ( x + 1 ) }[/tex] + [tex] 5^{x + 2}[/tex] - 3 = 0  

2 x [tex] 5^{2x}[/tex] . 5² + [tex] 5^{x}[/tex] . 5² - 3 = 0

2 x [tex] 5^{2x}[/tex] . 25 + [tex] 5^{x}[/tex] . 25 - 3 = 0

50 [tex] 5^{2x}[/tex] + 25[tex] 5^{x}[/tex] - 3 = 0

Dimisalkan y = [tex] 5^{x} [/tex]

50y² - 25y -3 = 0 => mencari nilai x dengan cara memfaktorkan  

( 5y + 3 ) ( 10y - 1 ) = 0

5y + 3  = 0 atau 10y - 1 = 0

5y = -3 atau 10y = 1

y = [tex]- \frac {3} {5}[/tex] atau y = [tex]\frac {1} {10}[/tex]

Ganti y menjadi [tex] 5^{x} [/tex] :

[tex] 5^{x} [/tex] = [tex]-\frac {3} {5}[/tex] atau [tex] 5^{x} [/tex] = [tex]\frac {1} {10}[/tex]

x = [tex]^{5}log ( -\frac {3} {5} )[/tex] atau x = [tex]^{5}log (\frac {1} {10} )[/tex]  

x = [tex]^{5}log ( -\frac {3} {5} )[/tex] tidak berlaku karena bernilai negatif.

Jadi, penyelesaian persamaan nya adalah x = [tex]^{5}log (\frac {1} {10} )[/tex]  

Kesimpulan

Terdapat persamaan berikut : 2 x [tex] 25^{x + 1}[/tex] + [tex] 5^{x + 2}[/tex] - 3 = 0 . Nilai x yg memenuhi persamaan berikut adalah [tex]^{5}log (\frac {1} {10} )[/tex]

31. 64(pangkat x) - 8(pangkat x+1) + 16 = 0


Eksponen[tex] {64}^{x} - {8}^{x + 1} + 16 = 0[/tex], maka nilai x adalah ⅔.

Eksponen adalah suatu bentuk operasi Matematika berjenis perkalian dengan bilangan yang sama kemudian diulang - ulang sesuai nominal pangkat yang tertera, atau bisa juga kita kenal sebagai pangkat atau nilai yang menunjukkan derajat kepangkatan suatu bilangan.

Eksponen seringkali digunakan oleh para peneliti untuk memudahkan dalam menulis angka. Seperti pengukuran jarak antar benda langit yang sangat sulit dihitung dengan satuan pengukuran biasa.

Dalam memahami eksponen, kita harus mengetahui terlebih dahulu sifat - sifat eksponen yang memiliki peran utama dalam dunia perpangkatan, antara lain :

[tex] {a}^{x} \times {a}^{y} = {a}^{x + y}[/tex]

[tex] {a}^{x} \div {a}^{y} = {a}^{x - y}[/tex]

[tex] {a}^{x} = \frac{1}{ {a}^{ - x} }[/tex] atau [tex] {a}^{x} = \frac{1}{ {a}^{ - x} } [/tex]

[tex] \sqrt[y]{ {a}^{x} } = {a}^{ \frac{x}{y} }[/tex]

[tex] {a}^{x} = {a}^{y}[/tex], maka x = y.

[tex] {x}^{a} = {y}^{a} [/tex], maka a = 0.

Agar lebih jelas dalam penerapan sifat - sifat eksponen di atas, simak pembahasan soal berikut.

PEMBAHASAN :

Perhatikan kembali soal di atas.

Kita diminta untuk menentukan nilai x dari

[tex] {64}^{x} - {8}^{x + 1} + 16 = 0[/tex]

Maka,

[tex]({8}^{2})^{x} - {8}^{x} + 16 = 0 \\ <=> {8}^{2x} - {8}^{x}.8 + 16 = 0[/tex]

Asumsikan [tex] {8}^{x} [/tex] = a. Maka,

[tex] {8}^{2x} - {8}^{x}.8 + 16 = 0 \\ <=> {a}^{2} - 8a + 16 = 0[/tex]

<=> (a - 4)(a - 4) = 0

<=> a = 4

a = [tex] {8}^{x} [/tex]

4 = [tex] {8}^{x} [/tex]

2² = [tex] {2}^{3x}[/tex]

3x = 2

x = ⅔

Jadi, untuk persamaan eksponen [tex] {64}^{x} - {8}^{x + 1} + 16 = 0[/tex], nilai x nya adalah ⅔.
