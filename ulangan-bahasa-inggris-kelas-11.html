Menghitung nilai rata - rata adalah :

= [tex]\frac {jumlah seluruh data} {banyaknya data}[/tex]

Jika menghitung jumlah seluruh data maka :

=  banyak nya data x nilai rata-rata siswa

Modus adalah data yg paling sering muncul.

1. Median untuk data genap  

>> Me = [tex]\frac {1} {2} ( x_{\frac {n} {2}} + x_{\frac {n} {2} + 1} )[/tex]

2. Median untuk data ganjil
