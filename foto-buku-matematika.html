Diagram lingkaran di samping menunjukkan file yang terdapat di dalam flashdisk milik reta yang berkapasitas 4 GB (setara dengan 4.000 MB). flashdisk tersebut diisi dengan file musik, foto, data buku ajar matematika, dan data lainnya.

A.    Jika Reta ingin menambahkan file data buku ajar baru yang berkapasitas 750 MB, apakah kapasitas flashdisk milik reta masih mencukupi?, jelaskan!. B.    Jika Reta tidak ingin menghapus file foto, file data buku ajar, dan file data lainnya di flashdisknya, berapa persen dari keseluruhan file musik yang harus dihapus agar data buku ajar baru dapat ditambahkan ke dalam flashdisk?

Statistika adalah ilmu (metode ilmiah) yang mempelajari cara-cara mengumpulkan, menyusun, menyajikan, dan menganalisis data serta cara mengambil kesimpulan yang logis sehingga dapat diambil keputusan yang akurat.

Penyajian data dalam bentuk

TabelDiagram batang Diagram garisDiagram lingkaran

Pembahasan

Perhatikan gambar diagram lingkaran yang ada pada lampiran.

A.  Menambahkan file data buku ajar baru berkapasitas 750 MB

Kapasitas kosong = [tex]\frac{10}{100}[/tex] × 4000 MB

                              = 400 MB

Ternyata flasdisk Reta tidak mencukupi apabila diisi dengan file data buku ajar baru berkapasitas 750 MB. Pada kapasitas flasdisk ruang kosong hanya tersedia 10% atau 400 MB, sehingga tidak dapat mencukupi file yg diisikan.

B.  Persen data yang dihapus dari keseluruhan file musik

Kekurangan data (dihapus) = kapasitas file baru - kapsitas ruang kosong

                                             = 750 MB - 400 MB

                                             = 350 MB

Kapasitas file musik = [tex]\frac{20}{100}[/tex] × 4000 MB

                                 = 800 MB

% file data yg dihapus dari keseluruhan file musik

= [tex]\frac{data~ dihapus}{file~ musik}[/tex] × 100 %

= [tex]\frac{350}{800}[/tex] × 100 %

= 43,75 %

Jadi file musik yang harus dihapus supaya mencukupi file baru adalah 43,75 % dari file musik

PENDAHULUAN AWAL :

Fungsi eksponen : f(x) = b^xFungsi linear : f(x) = bxFungsi pangkat : f(x) = x^b

POINT 01

Fungsi eksponen adalah fungsi yang memiliki bentuk dasar b^x dalam pemetaan x. Nilai fungsi (f(x)) akan setara dengan b^x. Misalnya.

Penjelasan dengan langkah-langkah:

Berikut adalah contoh bentuk rumus fungsi eksponen, linier, dan pangkat beserta penjelasan singkat:

1. Fungsi Eksponen:

a. Bentuk umum: f(x) = a^x, dengan a merupakan konstan positif yang disebut sebagai basis eksponen.

b. Contoh: f(x) = 2^x, f(x) = 3^x.

c. Penjelasan: Fungsi eksponen memiliki variabel berada di pangkat, dan nilai f(x) akan meningkat pesat ketika nilai x semakin besar.

2. Fungsi Linier:

a. Bentuk umum: f(x) = mx + c, dengan m merupakan kemiringan (slope) garis dan c merupakan nilai tetap (intersep) dari garis.

b. Contoh: f(x) = 2x + 3, f(x) = -0,5x + 7.

c. Penjelasan: Fungsi linier memiliki variabel x dengan pangkat 1 (linear) dan menghasilkan garis lurus ketika digambarkan pada koordinat.

3. Fungsi Pangkat:

a. Bentuk umum: f(x) = x^n, dengan x merupakan variabel dan n merupakan bilangan pangkat.

b. Contoh: f(x) = x^2, f(x) = x^3.

c. Penjelasan: Fungsi pangkat memiliki variabel berada di pangkat, dan nilai f(x) akan meningkat atau menurun dengan kecepatan yang bervariasi tergantung pada nilai n (bilangan pangkat).

Contoh-contoh di atas merupakan bentuk umum dari fungsi eksponen, linier, dan pangkat. Anda dapat menggunakan konstanta dan bilangan pangkat yang berbeda untuk menciptakan fungsi yang lebih spesifik sesuai dengan kebutuhan dan permasalahan yang diberikan dalam kurikulum matematika kelas 10, 9, 8, dan 7.
