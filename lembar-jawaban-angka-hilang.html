Penjelasan dengan langkah-langkah:

Soal diatas membahas mengenai Barisan Aritmatika. Rumus dasar Barisan aritmatika adalah Uₙ = a + (n-1)b.

Penyelesaian Soal:

1. Diketahui: Uₙ = 2n-1

  Ditanya: U₃₀ =

   Jawab: U₃₀ = 2n-1⇒ U₃₀ = 2(30)-1⇒ U₃₀ = 60-1⇒ U₃₀ = 59

Jadi Suku ke 30 barisan tersebut adalah 59

2. Diketahui: Uₙ = n(n-1) : 2

   Ditanya: U₈+ U₁₁=

   Jawab:

Uₙ = n(n-1) : 2

U₈ = 8(8-1) : 2⇒ U₈ = 8(7) : 2⇒ U₈ = 56 : 2⇒ U₈ = 28

U₁₁ = 11(11-1) : 2⇒ U₁₁ = 11(10) : 2⇒ U₁₁ = 110 : 2⇒ U₁₁ = 55

U₈+ U₁₁= 28+55 = 83

Jadi nilai dari U₈+ U₁₁ adalah 83

3. Diketahui: baris bilangan = 1, 8, 15, 22, ....

                     a = 1

                     b = 8-1 = 7

   Ditanya: U₄ = ?? U₅ = ?? U₆ = ??

  Jawab:

Uₙ = a + (n-1)b⇒Uₙ = 1 + (n-1)7⇒Uₙ = 1 +7n-7⇒Uₙ = 7n-6

Uₙ = 7n-6

U₄ = 7n-6 = 7(4)- 6 = 28 -6 = 22

U₅ = 7n-6 = 7(5)- 6 = 35 -6 = 29

U₆ = 7n-6 = 7(6)- 6 = 42 -6 = 36

Jadi tiga suku berikutnya adalah 22,29,36

4. Diketahui: Uₙ = (n+2) x 3

   Ditanya: U₁ = ?? U₂ = ?? U₃ = ??

   Jawab:

Uₙ = (n+2) x 3

U₁ = (n+2) x 3 ⇒U₁ = (1+2) x 3 ⇒U₁ = 3 x 3 ⇒U₁ = 9

U₂ = (n+2) x 3 ⇒U₂ = (2+2) x 3 ⇒U₂ = 4 x 3 ⇒U₂ = 12

U₃ = (n+2) x 3 ⇒U₃ = (3+2) x 3 ⇒U₃ = 5 x 3 ⇒U₃ = 15

Jadi tiga suku pertama dari barisan terebut adalah 9,12,15
