35. bahan penyedap makanan monosodium glutamat (msg) mempunyai susunan 13,6% na , 35,5% c ,4,8% h, 8,3% n , dan 37,8% o !!! tentukan rumus empirisdari msg


Diketahui :
Na = 13,6 %
C = 35,5%
H = 4,8%
N = 8,3%
O = 37,8%

Ditanya : Rumus empiris MSG ?

Jawab :

Misal Dalam 100 g MSG , terdapat 13,6 g Na ,35,5 g C , 4,8 g H , 8,3 g N , 37,8 g O.

Perbandingan mol :
= Na : C : H : N : O
= g/ Ar : g /Ar : g/Ar : g/Ar : g/Ar
= 13,6 / 23 : 35,5/12 : 4,8/1 : 8,3/14 : 37,8/16
= 0,59 : 2,95 : 4,8 : 0,59 : 2,36
= 1 : 5 : 8 : 1 : 4

Maka rumus empiris MSG adalah :
= Na : C : H : N : O
= 1 : 5 : 8 : 1 : 4
= NaC5H8NO4.

37. monosodium glutamar (MSG) adalah zat penyedap rasa yang lebih dikenal dengan nama vetsin. senyawa dengan massa molar sekitar 169g/mol tersebut mengandung 35,51%karbon 4,77% hidrogen 37,85% oksigen 8,29% nitrogen dan 13,60% natrium.. Tentukan rumus kimia senyawa tersebut


Mr MSG = 169 gr/mol
C = 35,51% = 35,51 gr
H = 4,77% = 4,77 gr
O = 37,85% = 37,85 gr
N = 8,29% = 8,29 gr
Na = 13,60% = 13,60 gr
perbandingan mol C : H : O : N : Na
= 35,51/12 : 4,77/1 : 37,85/16 : 8,29/14 : 13,60/23
= 2,96 : 4,77 : 2,366 : 0,592 : 0,591
= 5 : 8 : 4 : 1 : 1

rumus empirisnya = C5H8O4NNa
(C5H8O4NNa)n = 169
(12(5)+1(8)+16(4)+14+23)n = 169
169n = 169
n = 1

rumus senyawanya = C5H8O4NNa
