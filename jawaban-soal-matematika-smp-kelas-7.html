A={a,b,c,d,e}
B={a,c,e}
C={b,d,e,f,g}
Himpunan P merupakan himpunan bagian dari Q, jika setiap anggota P menjadi anggota Q. Ditulis P ⊂ Q.

B ⊂ A benar
A ⊂ C salah
B ⊂ C salah
{a} ⊂ A benar
{c,e,a} ⊂ {e,d,c,b,a}
∅ ⊂ C salah
Semangat!

A = {a, b, c, d, e}
B = {a, c, e}
C = {b, d, e, f, g}

Oleh karena semua anggota himpunan B termuat di himpunan A, maka
B ⊂ A → pernyataan a bernilai benar.

Oleh karena a dan c tidak termuat di himpunan C, maka A ⊄ C → pernyataan b bernilai salah.

Oleh karena a dan c tidak termuat di himpunan C, maka B ⊄ A → pernyataan c bernilai salah.

Oleh karena a adalah anggota himpunan A, maka {a} ⊂ A → pernyataan d bernilai benar.

Oleh karena himpunan kosong merupakan himpunan bagian dari sebarang himpunan tak kosong, maka pernyataan f bernilai benar.

NB: poin e tidak terlalu jelas tulisannya.
