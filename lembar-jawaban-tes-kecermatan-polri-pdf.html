Simpan sebuah file .xlsx menjadi .csv dan buka kembali file .csv tersebut dengan aplikasi pengolah lembar kerja. Simpan sebuah file .xlsx menjadi .pdf dan buka kembali file .pdf tersebut dengan aplikasi pengolah lembar kerja. Simpan sebuah file .xlsx menjadi .docx Apa yang terjadi? Apa kesimpulan kalian? Mengapa demikian?​


Percobaan dengan word processor atau pengolah kata:

Jika kita menyimpan sebuah file .docx menjadi .rtf dan membuka file .rtf tersebut, yang terjadi adalah file .docx akan dikonversi ke format .rtf, yang merupakan rich text format.  Jika kita menyimpan sebuah file .docx menjadi .pdf dan membuka file .pdf tersebut, yang terjadi adalah file .docx akan dikonversi ke format .pdf, yang merupakan format dokumen portabel (portable document format). Jika kita menyimpan sebuah file .docx menjadi .xlsx, dengan menulis langsung nama ekstensinya saat Save As. Buka file .xlsx tersebut, yang terjadi adalah file .docx akan disimpan dengan ekstensi .xlsx, tetapi tidak akan dikonversi ke format .xlsx, yang merupakan format spreadsheet (lembar kerja).

Pembahasan

Ada pun bila dilakukan percobaan dengan aplikasi spreadsheet atau pengolah lembar kerja:

Jika kita membuka sebuah file lembar kerja (.xlsx) dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan terbuka dengan normal, karena formatnya sesuai. Jika kita menyimpan sebuah file .xlsx menjadi .txt dan membuka kembali file .txt tersebut dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan dikonversi ke format .txt, yang merupakan format teks biasa (plain text). Jika kita simpan sebuah file .xlsx menjadi .csv dan membuka kembali file .csv tersebut dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan dikonversi ke format .csv, yang merupakan format comma separated values. Jika kita simpan sebuah file .xlsx menjadi .pdf dan membuka kembali file .pdf tersebut dengan aplikasi pengolah lembar kerja, yang terjadi adalah file .xlsx akan dikonversi ke format .pdf.Jika kita simpan sebuah file .xlsx menjadi .docx, yang terjadi adalah file .xlsx akan disimpan dengan ekstensi .docx, tetapi tidak akan dikonversi ke format .docx.Pelajari lebih lanjutMateri tentang beberapa aspek ilmu komputer https://brainly.co.id/tugas/16652646Materi tentang jenis software https://brainly.co.id/tugas/2300113Materi tentang beberapa software komputer https://brainly.co.id/tugas/6674111Detail Jawaban

27. misi mau nanya bagaimana cara membuat PDF dengan menggunakan WPS oofficesoalnya tugasnya disuruh buat cerpen nanti jawabannya dikirim menggunakan PDF


1. Mulai buat dokumen baru
Lalu buka aplikasi WPS Office yang sudah terpasang. Pada halaman muka kamu tinggal ketuk ikon "+". Kemudian kamu pilih opsi Dokumen.

2. Pilih jenis template dokumen
Dalam membuat dokumen baru di WPS Office, kamu bisa memilih tampilan kosong atau memilih berbagai template yang disediakan di dalamnya.

3. Mulai membuat PDF dari WPS Office
Lakukan pengeditan sesuai dengan kebutuhanmu dan untuk mulai menyimpan file, kamu tinggal ketuk ikon Save yang ada di pojok kiri atas.
Pada tampilan selanjutnya, kamu tinggal ketuk bagian Jalur untuk menyimpan file PDF di memori internal HP.

4. Pilih lokasi penyimpanan
Selanjutnya kamu tinggal pilih tempat akan menyimpan file, pilih Lokasi dan telusuri memori internal HP kamu.

5. Ubah format PDF
Terakhir kamu bisa mengubah nama dokumen sesuai dengan keinginan. Untuk cara membuat PDF, kamu tinggal ketuk menu dropdown pada pilihan format file dan pilih .pdf.
Jika sudah dirasa yakin, kamu tinggal ketuk Ekspor ke PDF dan tunggu hingga semua proses selesai.
