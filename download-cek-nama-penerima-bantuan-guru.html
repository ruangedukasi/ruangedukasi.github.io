Berikut adalah langkah-langkah untuk menyimpan (download) gambar dari internet dengan nama file "Gambar_1" dan menyimpannya dalam folder "Download Gambar" di drive D, yaitu:

Buka browser web dan kunjungi situs web tempat gambar yang ingin Anda download.Temukan gambar yang ingin Anda download dan pastikan Anda memiliki izin untuk mengunduhnya.Klik kanan pada gambar dan pilih opsi "Save Image As" (Simpan Gambar Sebagai) atau serupa, tergantung pada browser yang Anda gunakan.Sebuah jendela dialog akan muncul, memungkinkan Anda untuk memilih lokasi penyimpanan gambar.Arahkan jendela dialog ke drive D: dengan mengklik ikon drive di bagian atas jendela dan memilih "D:/".Buka folder "D:" dan buat folder baru dengan nama "Download Gambar" jika folder tersebut belum ada.Masuk ke folder "Download Gambar".Di jendela dialog, berikan nama file "Gambar_1" (tanpa tanda kutip) pada kolom "Nama file" atau "File name".Pastikan format file yang dipilih sesuai dengan format gambar yang ingin Anda download (misalnya, .jpg, .png, .gif).Klik tombol "Save" (Simpan) atau serupa di jendela dialog. Pembahasan

Download adalah proses pengambilan data atau file dari internet atau sumber lainnya dan menyimpannya ke perangkat lokal, seperti komputer, ponsel, atau tablet. Aktivitas download umumnya dilakukan untuk mendapatkan konten seperti gambar, video, musik, dokumen, perangkat lunak, dan berbagai jenis file lainnya.

Proses download melibatkan transfer data dari sumbernya ke perangkat pengguna melalui jaringan internet. Setelah file berhasil diunduh, file tersebut tersimpan dalam penyimpanan perangkat pengguna dan dapat diakses secara offline. Download sering dilakukan dengan menggunakan browser web, di mana pengguna dapat mengklik tautan atau gambar di situs web untuk memulai proses download.

30. Apa pengertian cek unjuk dan cek atas nama ??


Cek atas unjuk merupakan kebalikan dari cek atas nama. Di dalam cek atas unjuk tidak tertulis nama seseorang atau badan hukum tertentu jadi siapa saja dapat menguangkan cek atau dengan kata lain cek dapat diuangkan oleh si pembawa cek. Sebagai contoh di dalam cek tersebut tertulis bayarlah tunai, atau cash atau tidak ditulis kata-kata apa pun.
Merupakan cek yang diterbitkan atas nama seseorang atau badan hukum tertentu yang tertulis jelas di dalam cek tersebut. Sebagai contoh jika di dalam cek tertulis perintah bayarlah kepada: Tn. Roy Akase sejumlah Rp 3.000.000,- atau bayarlah kepada PT. Marindo uang sejumlah Rp 1.000.000,- maka cek inilah yang disebut dengan cek atas nama, namun dengan catatan kata "atau pembawa" dibelakang nama yang diperintahkan dicoret
