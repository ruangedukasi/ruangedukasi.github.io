1. tolong ya kak dapat tugas tambahan dari guru​


Sebelum masuk ke bagan persilangan, terlebih dahulu tentukan fenotipe dan genotipe dari indukan, diketahui sebagai berikut :

Kedelai kuning : KKKedelai hijau : kk

Setelah genotipe dan fenotipe indukan diketahui maka buat bagan persilangannya :

P         :  Kedelai kuning >< Kedelai hijau

                   (KK)                        (kk)

Gamet :         K                            k

F1        :              Kedelai kuning

                                (Kk)

Untuk mencari F2 maka silangkan F1 dengan sesamanya :

P2       :  Kedelai kuning >< Kedelai kuning

                   (Kk)                      (Kk)

Gamet :     K, k                       K, k

F2       :  1). KK --> Kedelai kuning

             2). Kk --> Kedelai kuning

             3). Kk --> Kedelai kuning

             4). kk --> Kedelai hijau

Nah, pada F2 ternyata dihasilkan 4 anakan dengan perbandingan sebagai berikut :

Rasio Fenotipe (RF) :

= Kedelai kuning : Kedelai hijau

=  3     :     1

= 75% : 25%

Rasio Genotipe (RG) :

= KK  :   Kk   : kk

=  1    :    2    :   1

= 25% : 50% : 25%

Kesimpulan :

a). Parental Kedelai kuning >< Kedelai hijau

b). F1 = Kedelai kuning

c). Sifat persilangan adalah monohibrid dominan penuh.

d). ● Parental = Bulat kuning >< Bulat kuning

    ● Gamet =       K, k                  K, k
