12. Word 2007 mampu menyimpan file ke dalam format lain yaitu​


Doc dan docx adalah dua format file atau ekstensi file yang digunakan di aplikasi Microsoft Word, salah satu aplikasi yang terdapat di paket aplikasi Microsoft Office. Format file doc digunakan di Microsoft Word 2003 dan versi sebelumnya, sedangkan docx digunakan di aplikasi Microsoft Word 2007 dan versi sesudahnya. Salah satu kendala yang ada sementara ini untuk format file terbaru (docx) adalah terkait kompatibilitas.

30. mengapa format MP3 lebih banyak dipilih daripada format audio lain?​


MP3 atau MPEG Layer 3, merupakan File Audio yang sudah dikompresi. Kualitasnya sangat baik hingga mencapai resolusi 16-bit. Kualitas ini mendekati kualitas CD Audio dengan konfigurasi stereo 2.1.

Keuntungan utama dari MP3 adalah ukuran filenya yang kecil. Satu menit Audio rata-rata hanya berukuran 1 MB. Bitrate-nya pun bisa dipilih sesuai kebutuhan, mulai dari 96, 128, 160, 192 hingga 256 Kbps.
